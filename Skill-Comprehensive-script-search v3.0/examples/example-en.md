# Example: English-title Script Search

> Demonstrates the workflow for finding English-language scripts (Shakespeare, Miller, etc.).

## Scenario 1: Find the full text of *Hamlet*

### User input
```
Find Shakespeare's Hamlet full text PDF, save to ~/Desktop/scripts
```

### Skill execution steps

#### Step 1: Parse the request
```yaml
play_name: Hamlet
author: Shakespeare
language: en
save_path: ~/Desktop/scripts
analyze: true
format: pdf
```

#### Step 2: Engine dispatch (English-first)

Calls the 9 international engines:
- Google: `https://www.google.com/search?q=Hamlet+Shakespeare+full+text+filetype:pdf`
- Google HK: `https://www.google.com.hk/search?q=Hamlet+filetype:pdf`
- DuckDuckGo: `https://duckduckgo.com/html/?q=Hamlet+Shakespeare+site:archive.org`
- Yahoo: `https://search.yahoo.com/search?p=Hamlet+Shakespeare+script`
- Startpage: `https://www.startpage.com/sp/search?query=Hamlet+filetype:pdf`
- Brave: `https://search.brave.com/search?q=Hamlet+Shakespeare+text`
- Ecosia: `https://www.ecosia.org/search?q=Hamlet+filetype:pdf`
- Qwant: `https://www.qwant.com/?q=Hamlet+Shakespeare`
- WolframAlpha: `https://www.wolframalpha.com/input?i=Hamlet+Shakespeare+play`

Plus 7 Chinese engines (looking for Chinese translations):
- Baidu: `https://www.baidu.com/s?wd=哈姆雷特+莎士比亚+中文+剧本+filetype:pdf`
- etc.

#### Step 3: Filter links

Top-priority targets:
- ⭐⭐⭐⭐⭐ archive.org
- ⭐⭐⭐⭐⭐ gutenberg.org (Shakespeare's 37 plays)
- ⭐⭐⭐⭐ openlibrary.org

#### Step 4: Download

```bash
# From Project Gutenberg
./scripts/download.sh "https://www.gutenberg.org/files/1525/1525-h/1525-h.htm" "$HOME/Desktop/scripts" "Hamlet_Shakespeare.html"

# Or from Internet Archive
./scripts/download.sh "https://archive.org/download/hamlet_2403_librivox/hamlet.pdf" "$HOME/Desktop/scripts" "Hamlet_Shakespeare.pdf"
```

#### Step 5: Analysis report

```markdown
# 🎭 *Hamlet* Analysis Report

═══════════════════════════════════

📊 Word count: ~30,000 words

- Core proposition: action vs. inaction, the nature of existence
- Values: critique of indecision, exploration of mortality

## [Characters]
| Character | Role | Core trait | Arc |
|-----------|------|-----------|-----|
| Hamlet | Prince of Denmark | Philosophical, indecisive | Grief → revenge → acceptance |

- Act 1: exposition — the Ghost appears, Hamlet learns the truth
- Act 2: development — Polonius, Ophelia, "fishmonger"

  - Hamlet vs. himself (inaction)
- Internal conflicts:
  - Hamlet: action vs. contemplation
  - Claudius: guilt vs. power
  - Ophelia: love vs. duty
- Core contradiction: the paralysis of overthinking
- Turning point: Act 3 Scene 3 — Hamlet spares Claudius

## [Style]

- Language: iambic pentameter, mixed with prose
- Structure: classical five-act, Aristotelian
- Aesthetic: Renaissance humanism
```
