# Example: Chinese-title Script Search

> Demonstrates the workflow for finding Chinese-language scripts (Cao Yu, Lao She, etc.).

## Scenario 1: Find *Thunderstorm* (雷雨)

### User input
```
Find me a copy of 雷雨 (Thunderstorm) and save it to ~/Desktop/caoyu
```

### Skill execution steps

#### Step 1: Parse the request
```yaml
play_name: 雷雨
author: 曹禺
language: zh
save_path: ~/Desktop/caoyu
analyze: true
```

#### Step 2: Engine dispatch (Chinese-first)

Calls the 7 Chinese engines:
- Baidu: `https://www.baidu.com/s?wd=雷雨+剧本+filetype:pdf`
- Bing CN: `https://cn.bing.com/search?q=雷雨+剧本&ensearch=0`
- Bing INT: `https://cn.bing.com/search?q=雷雨+剧本&ensearch=1`
- 360: `https://www.so.com/s?q=雷雨+剧本`
- Sogou: `https://sogou.com/web?query=雷雨+剧本`
- WeChat: `https://wx.sogou.com/weixin?type=2&query=雷雨+剧本`
- Shenma: `https://m.sm.cn/s?q=雷雨+剧本`

Plus 9 international engines (looking for English translations or academic editions):
- Google: `https://www.google.com/search?q=雷雨+曹禺+script+filetype:pdf`
- DuckDuckGo: `https://duckduckgo.com/html/?q=雷雨+Thunderstorm+script`
- etc.

#### Step 3: Filter links (sorted by reliability)

```
⭐⭐⭐⭐⭐ Internet Archive
⭐⭐⭐⭐ Daoke Baba, Taodocs
⭐⭐⭐ Original Force, Renren Library
⭐ Baidu Wenku (mostly paid)
```

#### Step 4: Download

```bash
./scripts/download.sh "https://archive.org/download/leiyu/leiyu.pdf" "$HOME/Desktop/caoyu" "雷雨_曹禺_剧本.pdf"
```

#### Step 5: Analysis report

```markdown
# 🎭 *Thunderstorm* (雷雨) Analysis Report

═══════════════════════════
📁 File: ~/Desktop/caoyu/雷雨_曹禺_剧本.pdf
📊 Word count: ~58,000
📜 Acts: 4
═══════════════════════════

## [Theme]
- Core proposition: the collapse of feudal family and the inevitability of fate
- Values: critique of feudal autocracy and hypocritical morality
- Era: 1920s–30s semi-colonial, semi-feudal China
- Key imagery: thunderstorm, cicada song, the oppressive Zhou mansion

## [Characters]
| Role | Identity | Core trait | Arc |
|------|----------|-----------|-----|
| Zhou Puyuan | Master of the Zhou mansion | Autocratic, hypocritical | Authority → collapse |
| Fan Yi | Zhou Puyuan's wife | Rebellious, passionate | Repression → eruption |
| Zhou Ping | Zhou Puyuan's eldest son | Weak, conflicted | Escape → suicide |
| Si Feng | Lu Ma's daughter | Innocent | Innocence → tragedy |
| Lu Shiping | Zhou Puyuan's old lover | Resilient | Endurance → awakening |
| Lu Gui | Steward | Self-serving | Always self-serving |
| Zhou Chong | Zhou Puyuan's younger son | Idealistic | Ideal → disillusionment |

## [Acts & Scenes]
- Act I: drawing room of the Zhou mansion, character relations introduced
- Act II: Zhou mansion, conflicts escalate
- Act III: Lu Gui's home, contrast with the lower class
- Act IV: Zhou mansion, the tragedy erupts
- Estimated runtime: ~3 hours

## [Conflict]
- External conflict:
  - Zhou Puyuan vs Fan Yi (control vs rebellion)
  - Zhou Ping vs Fan Yi (emotional entanglement)
  - Zhou family vs Lu family (class opposition)
- Internal conflict:
  - Zhou Ping: weakness vs responsibility
  - Fan Yi: love intertwined with hate
- Core contradiction: feudal ethics crushing human nature
- Turning point: the end of Act III, when Shiping and Zhou Puyuan meet again

## [Style]
- School: realism (influenced by Ibsen)
- Language: stage dialogue, highly literary
- Structure: traditional four-act, closed structure
- Aesthetic: realism with symbolism (the thunderstorm symbolizing the coming storm)

═══════════════════════════
📝 Analysis time: 2026-06-12
🤖 Auto-generated by the find-script Skill
═══════════════════════════
```

## Scenario 2: Find *Teahouse* (茶馆)

### User input
```
Download Lao She's 茶馆 (Teahouse) script
```

### Execution flow

#### Search
- Baidu: `老舍 茶馆 剧本 filetype:pdf`
- Sogou: `site:doc88.com 茶馆 老舍`
- etc.

#### Download
```bash
./scripts/download.sh "URL" "$HOME/Desktop/laoshe" "茶馆_老舍_剧本.pdf"
```

#### Analysis brief
```markdown
# 🎭 *Teahouse* (茶馆) Analysis Report

[Theme]
- Core proposition: the fate of small people in a turbulent era
- Era: half a century of China, 1898–1948

[Characters]
- Wang Lifa: teahouse owner, trying to please everyone
- Chang Sye: upright Manchu bannerman
- Qin Er: would-be industrialist
- The three span three acts, witnessing the changing times

[Acts & Scenes]
- Three acts across 50 years — an innovative structure

[Conflict]
- External conflict: the turmoil of the times
- Internal conflict: small people's sense of powerlessness

[Style]
- School: realism
- Structure: "portrait-exhibition" form — Lao She's invention
- Feature: 7000 words capture 50 years
```
