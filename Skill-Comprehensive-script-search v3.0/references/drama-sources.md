## Recommended by Type

Script sources differ significantly across types. Below are the **best starting points** for each:

| Type | First-choice source | Backup sources | Default copyright |
|------|---------------------|----------------|-------------------|
| Stage play | archive.org / gutenberg.org | doc88.com / Drama Online | PD |
| Opera | ctext.org / imslp.org | Chinese-opera networks / doc88.com | PD (classical) / unknown (modern) |
| Film | archive.org (pre-1929) | IMSDb / Script Slug / Script-O-Rama | PD (early) / Copyrighted (modern) |
| Television | archive.org (1950s) | doc88.com (user upload) | Copyrighted |

## Stage play (Play)

### Internet Archive (archive.org) ⭐⭐⭐⭐⭐

- **URL**: https://archive.org
- **Strength**: massive catalog of public-domain scripts.
- **Formats**: PDF, TXT, HTML, EPUB.
- **Access**: direct download, no login.
- **Notable works**:
  - Shakespeare complete works
  - Greek tragedies
  - 20th-century classics
  - Ibsen / Chekhov / Strindberg

### Project Gutenberg (gutenberg.org) ⭐⭐⭐⭐⭐

- **URL**: https://www.gutenberg.org
- **Strength**: 70,000+ free e-books, including many classical plays.
- **Formats**: TXT, HTML, EPUB, MOBI.
- **Access**: direct download.
- **Notable works**:
  - Shakespeare's 37 plays
  - Oscar Wilde's complete works
  - Ibsen / Chekhov / Strindberg

### Open Library (openlibrary.org) ⭐⭐⭐⭐

- **URL**: https://openlibrary.org
- **Strength**: Internet Archive sub-project.
- **Formats**: PDF, EPUB.
- **Access**: some titles require borrowing.

### Drama Online ⭐⭐⭐⭐

- **URL**: https://www.dramaonlinelibrary.com
- **Strength**: academic-publisher collection (well-edited).
- **Access**: institutional subscription required.

## Opera / Xiqu / Musical

### Chinese Text Project (ctext.org) ⭐⭐⭐⭐⭐

- **URL**: https://ctext.org
- **Strength**: classical Chinese opera scripts in **fully public-domain** form (with gongche / numbered notation).
- **Formats**: HTML, TXT.
- **Access**: direct download.
- **Notable works**:
  - Yuan zaju: *Romance of the Western Chamber*, *The Injustice to Dou E*, *The Peony Pavilion*, *The Palace of Eternal Life*, *The Peach Blossom Fan*.
  - Peking-opera traditional repertoire.
  - Kunqu aria scores.
- **Search tip**: `site:ctext.org Peony Pavilion libretto`.

### IMSLP (imslp.org) ⭐⭐⭐⭐⭐

- **URL**: https://imslp.org
- **Strength**: international sheet-music library; PD scores for Western opera / musical.
- **Formats**: PDF, MIDI.
- **Access**: direct download.
- **Notable works**:
  - Mozart / Verdi / Puccini / Wagner full scores.
  - Bizet *Carmen* / Puccini *La Bohème*.
  - Classical-musical libretti.

### Chinese-opera networks ⭐⭐⭐⭐

- **URL**: http://www.zhongguoxiqu.net
- **Strength**: Peking opera / Yue opera / Yu opera / Huangmei opera and other traditional repertoires.
- **Format**: HTML.
- **Notable works**: classical and adapted scripts.

### Chinese Peking-Opera Art Network ⭐⭐⭐

- **Strength**: focus on Peking-opera scripts, lyrics, and scores.
- **Use case**: finding specific Peking-opera excerpt scenes.

### doc88.com / taodocs.com ⭐⭐⭐

- **Use case**: modern Chinese-opera adaptations (e.g. newly composed Peking opera, regional operas).
- **Copyright**: mostly `user_uploaded`; verify yourself.

## Film

### Internet Archive (archive.org) ⭐⭐⭐⭐⭐ — early public-domain films

- **URL**: https://archive.org
- **Strength**: **almost all pre-1929 silent films are PD** (Chaplin / Keaton / Griffith era).
- **Notable works**:
  - Charlie Chaplin: *City Lights*, *Modern Times*, *The Great Dictator*.
  - Buster Keaton: *The General*, *Sherlock Jr.*.
  - D. W. Griffith: *The Birth of a Nation*.
- **Search tip**: `site:archive.org silent film script`.

### IMSDb (Internet Movie Script Database) ⭐⭐⭐⭐

- **URL**: https://imsdb.com
- **Strength**: modern screenplays **for study and reference only**.
- **Formats**: HTML, PDF.
- **Notable works**: many Hollywood screenplays (mostly under copyright).
- **Copyright note**: ⚠️ the vast majority are still under copyright.

### Script Slug ⭐⭐⭐⭐

- **URL**: https://thescriptslug.com
- **Strength**: large collection of contemporary screenplays (study use).
- **Copyright**: ⚠️ copyrighted.

### Script-O-Rama ⭐⭐⭐⭐

- **URL**: https://script-o-rama.com
- **Strength**: older + some new screenplays.
- **Notable works**: 1990s classics.

### Drew's Script-O-Rama ⭐⭐⭐⭐

- **URL**: https://www.drewsmoviescripts.com
- **Strength**: academically-oriented organization.
- **Use case**: looking for a specific era / genre.

### Hong Kong / Taiwan / Chinese cinema ⭐⭐⭐

- **Sources**: doc88.com / taodocs.com (user uploads).
- **Notable works**: Wong Kar-wai / Ang Lee / Hou Hsiao-hsien / Edward Yang screenplays.
- **Copyright**: ⚠️ almost all copyrighted; user authorization required.

### Chinese-film materials ⚠️ copyright warning

- Pre-1949 Chinese film scripts: author has been dead 70+ years, may be PD.
- 1949–2000: most still under copyright.
- Post-2000: almost all copyrighted.

## Television (TV)

### ⚠️ Strong warning

> **Modern TV (post-2000) is almost all still under copyright.**
> Post-1990 US / UK / Japanese / Korean / Chinese drama — `copyright=copyrighted`.

### Internet Archive (archive.org) ⭐⭐⭐ — early public-domain shows

- **URL**: https://archive.org/details/tv
- **Strength**: 1950s black-and-white shows, early public-domain TV.
- **Notable works**:
  - *I Love Lucy* (1951–1957) — most episodes PD.
  - *The Twilight Zone* (older version, partial PD).
  - 1950s American live TV.
- **Search tip**: `site:archive.org 1950s television script`.

### Wikipedia: "Public domain television" ⭐⭐⭐

- **URL**: https://en.wikipedia.org/wiki/Public_domain_television
- **Strength**: list of TV shows that have entered the public domain.
- **Use case**: verifying PD status.

### doc88.com / taodocs.com ⭐⭐ — user uploads

- **Notable works**: modern Chinese TV scripts, mostly user uploads.
- **Copyright**: ⚠️ `user_uploaded` — **for study and reference only**.
- **Verification**: when `copyright=user_uploaded`, you must verify the uploader's authorization.

### Academic / educational excerpts ⭐⭐⭐

- **Scope**: quotes within fair-use bounds.
- **Use case**: teaching, criticism, research.
- **⚠️ Not a substitute for downloading the full script.**

## Chinese Platforms ⭐⭐⭐

### Daoke Baba (doc88.com)

- **Strength**: general document sharing.
- **Formats**: PDF, DOC, PPT.
- **Access**: some free, some require points.
- **Notable works**: modern Chinese stage plays, modern opera, Hong Kong / Taiwan film scripts.

### Taodocs (taodocs.com)

- **Strength**: education-focused resources.
- **Formats**: PDF, DOC.
- **Access**: some free.

### Original Force (max.book118.com)

- **Strength**: general documents.
- **Formats**: PDF, DOC, PPT.
- **Access**: some free.

### Renren Library (renrendoc.com)

- **Strength**: academic and humanities resources.
- **Formats**: PDF, DOC.
- **Access**: some free.

### Baidu Wenku (wenku.baidu.com)

- **Strength**: largest catalog.
- **Formats**: PDF, DOC, TXT.
- **Access**: most require VIP.
- **Alternative**: look for mirrors on other platforms.

## Discouraged Platforms ❌

- **Qidian** — web novels, not scripts.
- **JJWXC** — fan-fiction / romance.
- **Taobao** — pirated resources, copyright risk.
- **Netdisk search engines** — high infringement rate.
- **Unauthorized modern TV / film sites** — all copyrighted.

## Copyright Verification Checklist

> ⚠️ **Verify the copyright manually in the following cases:**

1. Any result with `copyright=unknown`.
2. **Modern film / TV** (even if the source looks legitimate).
3. **Modern opera adaptations** (e.g. newly composed Peking opera).
4. **Modern Chinese stage plays** (e.g. Meng Jinghui, Liu Dehua — contemporary playwrights).
5. **User-uploaded resources** (doc88 / taodocs) — check whether the uploader marks authorization.
