# Film — 5-Dim Analysis Framework

> find-script v3.0.0 / type=film.
> Covers: feature films, shorts, documentaries, web films.

## When to use

- Early public-domain films: Charlie Chaplin *City Lights*, *Modern Times*, *The Great Dictator*.
- Buster Keaton: *The General*, *Sherlock Jr.*.
- D. W. Griffith: *The Birth of a Nation*.
- Silent / early sound films (almost all PD pre-1929).
- Modern films (require user authorization, copyright tag = `copyrighted`).

## The 5 Dimensions

### Dimension 1: Theme

- Core proposition: the fundamental question the film explores.
- Values: positive / negative / neutral / ambiguous.
- Historical context: historical / social / political setting.
- Key imagery: recurring visual / auditory symbols.

### Dimension 2: Characters

| Field | Description |
|-------|-------------|
| Name | Actor + role |
| Role | Lead / supporting / ensemble |
| Core traits | Personality + appearance |
| Character arc | Change / Steadfast |
| Function | Plot driver / thematic vehicle / POV character |

### Dimension 3: Scenes

- Unit: scene / shot / sequence.
- Markers: INT. / EXT. / FADE IN / FADE OUT / CUT TO.
- Duration: scene → segment → full film.

> Detection regex: `INT\.` / `EXT\.` / `FADE IN` / `CUT TO` and other format markers.

### Dimension 4: Conflict

- External conflict: person vs person / person vs society / person vs fate / person vs technology / person vs nature.
- Internal conflict: character's psychological struggle / moral dilemma / identity.
- Core contradiction: the fundamental conflict driving the whole film.
- Turning points: which scene / which shot / Plot Point I / II.

### Dimension 5: Audiovisual + Genre

#### Camera

- Shot size: Extreme Long Shot (ELS) / Long Shot (LS) / Medium (MS) / Close-up (CU) / Extreme Close-up (ECU).
- Camera movement: Pan / Tilt / Tracking / Dolly / Crane / Handheld / Steadicam.
- Angle: POV / Over-shoulder / Bird's-eye / Worm's-eye.
- Composition: Rule of thirds / Symmetry / Depth of field.

#### Editing

- Continuity editing.
- Jump cut.
- Montage (Soviet school).
- Parallel / cross-cutting.
- Long take.
- L-cut / J-cut / match cut.

#### Sound design

- Voice-over.
- Score / Soundtrack.
- Silence.
- Ambient sound.
- SFX.
- Foley.

#### Genre

- Western.
- Film Noir.
- Musical.
- Sci-Fi.
- Thriller.
- Horror.
- Romance.
- War.
- Comedy.
- Biopic.
- Epic.
- Wuxia.
- Animation.

## Copyright Notes

⚠️ **Early PD is fair game by default**: pre-1929 American films (Chaplin / Keaton / Griffith, etc.) + international authors more than 70 years deceased.
⚠️ **Modern films only with user authorization**: when the search result is `copyright=copyrighted`, use only within the LICENSE scope.
⚠️ Chaplin's later works (1940s+) are still under copyright and need case-by-case verification.
⚠️ Most pre-1949 Chinese films have authors deceased 70+ years, so may be PD.
