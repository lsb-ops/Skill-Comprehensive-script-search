# Stage Play — 5-Dim Analysis Framework

> find-script v3.0.0 / type=play (default; backward compatible with v1.4 find-stage-play).

## When to use

- Western classical drama (Shakespeare / Ibsen / Chekhov / O'Neill / Arthur Miller).
- Modern Chinese stage plays (Cao Yu / Lao She / Tian Han / Xia Yan / Meng Jinghui).
- Modern experimental / small-theatre / one-person shows.

## The 5 Dimensions

### Dimension 1: Theme

- Core proposition: the most fundamental question the play explores (one sentence).
- Values: positive / negative / neutral.
- Historical context: specific era, social features.
- Key imagery: recurring symbols.

### Dimension 2: Characters

| Field | Description |
|-------|-------------|
| Name | Chinese + original |
| Role | Lead / supporting / extra |
| Core traits | Personality tags |
| Character arc | Opening → closing: growth / fall / flat |
| Function | Plot driver / thematic symbol / foil |

### Dimension 3: Acts & Scenes

- Total acts N
- Total scenes M
- Estimated runtime X hours
- Core event per act (one sentence)

> Detection regex: `第N幕` / `第N场` / `Act N` / `Scene N`.

### Dimension 4: Conflict

- External conflict: person vs person / person vs society / person vs fate.
- Internal conflict: a character's psychological struggle.
- Core contradiction: the fundamental conflict driving the whole play.
- Turning points: which act / scene.

### Dimension 5: Style

- School: realism / romanticism / symbolism / absurdism / epic / cruelty.
- Language: prose / poetic / dialect / monologue.
- Structure: traditional three-act / open / linear / non-linear / circular.
- Aesthetic: realism / symbolism / absurdism.

## School Cheat Sheet

| School | Representative work | Characteristics |
|--------|--------------------|-----------------|
| Classicism | Racine *Andromaque* | Three unities, reason, elegance. |
| Romanticism | Hugo *Hernani* | Strong emotion, exotic flavor. |
| Realism | Ibsen *A Doll's House* | Everyday language, social issues. |
| Symbolism | Maeterlinck *The Blue Bird* | Metaphor, mystery, poetry. |
| Expressionism | Kaiser *From Morn to Midnight* | Inner experience externalized, exaggeration. |
| Theatre of the Absurd | Beckett *Waiting for Godot* | Nihilism, circular, illogical. |
| Epic theatre | Brecht *Mother Courage* | Alienation effect, political. |
| Theatre of Cruelty | Artaud *The Cenci* | Ritual, violence, sensory shock. |
