# Television — 5-Dim Analysis Framework

> find-script v3.0.0 / type=tv.
> Covers: TV series, web dramas, miniseries, animated series, anthology shows.

## When to use

- Classic US TV scripts (some PD): old shows / early series.
- Classic UK PD: early BBC works.
- Web dramas / short dramas / vertical-screen dramas.
- Animated series.
- Anthology (each episode a standalone story).

## The 5 Dimensions

### Dimension 1: Theme

- Core proposition: the fundamental question the whole show explores.
- Values: positive / negative / neutral / complex.
- Historical context: story setting + production era.
- Key imagery: visual / auditory symbols running through the show.

### Dimension 2: Characters

| Field | Description |
|-------|-------------|
| Name | Main + guest |
| Role | Lead / supporting / guest star / ensemble |
| Core traits | Personality + relationship network |
| Character arc | Season-wide arc + per-episode mini-arc |
| Function | POV / antagonist / emotional core / comic relief |

### Dimension 3: Episodes & Scenes

- Unit: Episode + Scene.
- Markers: EPISODE N / S01E01 / COLD OPEN / ACT N / TAG.
- Internal structure: COLD OPEN → ACT 1–4 (or 5) → TAG.
- Duration: 22 min / 44 min / 60+ min per episode.

> Detection regex: `EPISODE N` / `S\d+E\d+` / `COLD OPEN` / `INT\.` / `EXT\.` / `CUT TO`.

### Dimension 4: Conflict

- External conflict: family / workplace / society / enemy forces / rival organizations / supernatural.
- Internal conflict: character psychology / identity / moral dilemma.
- Core contradiction: the season's main conflict.
- Turning points: Pilot hook / Mid-season finale / Season finale / Series finale.

### Dimension 5: Series Structure + Genre

#### Series structure

- **Procedural**: each episode is a self-contained story, with the main plot advancing slowly.
  - Example: *CSI*, *House M.D.*
- **Multi-arc**: multiple storylines interweave.
  - Example: *Lost*, *Westworld*.
- **Serial / Soap**: each episode picks up right after the previous; must be watched in order.
  - Example: *Breaking Bad*, *Game of Thrones*.
- **Anthology**: each episode has independent characters / stories, but shares a theme.
  - Example: *Black Mirror*, *The Twilight Zone*.
- **Limited Series**: a single season tells the whole story.
  - Example: *Chernobyl*, *The Heritage*.
- **Hybrid**: a mix of the above.

#### Season structure

- Seasonal — traditional TV.
- Continuous — streaming.
- Limited — miniseries.
- Split season — a season split into two halves.

#### Genre

- Drama.
- Comedy / Sitcom.
- Crime / Detective.
- Medical.
- Sci-Fi.
- Fantasy.
- Period.
- War.
- Legal.
- Thriller.
- Horror.
- Romance.
- Animation.
- Wuxia / Xianxia.
