# Opera (Chinese Opera / Western Opera / Musical) — 5-Dim Analysis Framework

> find-script v3.0.0 / type=opera.
> Covers: Chinese opera (Peking opera / Yue opera / Huangmei opera / Kunqu / Sichuan opera / Yu opera / Qinqiang / Pingju / Cantonese opera), Western opera, musical.

## When to use

- Peking opera: *The Drunken Concubine*, *Farewell My Concubine*, *Suo Lin Nang*.
- Yue opera: *Dream of the Red Chamber*, *Butterfly Lovers*.
- Huangmei opera: *The Heavenly Match*, *The Female Prince*.
- Kunqu: *The Peony Pavilion*, *The Palace of Eternal Life*.
- Western opera (Puccini *Tosca*, Verdi *La Traviata*).
- Musical (*Cats*, *The Phantom of the Opera*, *Hamilton*).

## The 5 Dimensions

### Dimension 1: Theme

- Core proposition: the fundamental question the libretto / score explores.
- Values: traditional ethics / revolutionary theme / secular emotion / religious redemption.
- Historical context: dynasty / century / cultural milieu.
- Key imagery: typical opera motifs (love tragedy, loyalty vs treachery, family rise and fall).

### Dimension 2: Characters

- Role types (Chinese opera): sheng / dan / jing / mo / chou.
  - Subtypes: laosheng / xiaosheng / wusheng / qingyi / huadan / wudan / laodan / hualian / choujue.
- Role types (Western opera): tenor / soprano / baritone / lyric / coloratura.
- Character arc: Chinese opera keeps the role-type fixed (focus on movement / vocal change); Western opera can be more modern.

### Dimension 3: Scenes / Folds

- Chinese opera unit: zhe / chu / chang / mu.
- Western opera unit: Act + Scene.
- Musical unit: Act + Scene + song number.

> Detection regex: `第N场` / `第N折` / `第N出` / `\[aria name\]` / `\[xipi\]` / `\[erhuang\]`.

### Dimension 4: Conflict

- External conflict:
  - Chinese opera: dynastic change / loyalty vs treachery / family feud / love tragedy.
  - Western opera: class / faith / war / individual vs fate.
  - Musical: identity / class / self-actualization.
- Internal conflict: character's inner world / moral dilemma / emotion vs duty.
- Core contradiction: a one-sentence summary.
- Turning points: which scene / which aria.

### Dimension 5: Vocal Style & Music

#### Chinese opera

- **Role types**: sheng / dan / jing / mo / chou.
- **Vocal systems**:
  - Peking opera: xipi / erhuang.
  - Kunqu: arias (zaoluo pao / shanpo yang / duan zheng hao / gunxiu qiu).
  - Yue opera: chi diao / xianxia diao / sigong diao.
  - Yu opera: Yu-dong / Yu-xi diao.
  - Huangmei opera: huaqiang / pingci / caiqiang.
- **Beat patterns**: yuanban / manban / kuaiban / jiban / sanban / yaoban / daoban / liushui / erliu.
- **Lyric characteristics**: poetic / classical Chinese / colloquial / dialect.

#### Western opera

- **Voice types**: Soprano / Mezzo / Contralto / Tenor / Baritone / Bass.
- **Forms**: Aria / Recitative / Ensemble / Chorus.
- **Compositional schools**: Verismo / Bel canto / Singspiel.

#### Musical

- **Song types**: Ballad / Up-tempo / Character song / Production number.
- **Structure**: Book musical / Concept musical / Jukebox musical / Revue.

⚠️ Most musicals after 1940 are still under copyright.
