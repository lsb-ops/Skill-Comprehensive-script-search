## Asking a Question

Before opening an issue, please:

1. Read the [FAQ.md](FAQ.md)
2. Search [existing issues](https://github.com/yourname/find-script/issues)
3. Try the latest version (your bug may already be fixed)

If none of the above resolve your question, open a
[GitHub Discussion](https://github.com/yourname/find-script/discussions) (Q&A
category) rather than an issue.

## Filing a Bug

Open an issue at
[github.com/yourname/find-script/issues/new?template=bug_report.md](https://github.com/yourname/find-script/issues/new?template=bug_report.md).

Please include:

- find-script version (`./scripts/find-play.sh --version` or check
  `metadata.json`)
- Operating system and version
- Shell (`bash --version`, `zsh --version`)
- Steps to reproduce
- Actual vs. expected behavior
- Relevant log output or error messages

## Requesting a Feature

Open a feature request at
[github.com/yourname/find-script/issues/new?template=feature_request.md](https://github.com/yourname/find-script/issues/new?template=feature_request.md).

Please describe:

- The problem you are trying to solve
- Your proposed solution
- Any alternatives you have considered
- Whether you would be willing to submit a PR

## Response Times

This is an open-source project maintained in spare time. Please be patient —
typical response times are 7-14 days. Critical bugs and security issues are
prioritized; see [SECURITY.md](SECURITY.md) for the security contact.

## Commercial Support

custom development, please contact the maintainer directly.
