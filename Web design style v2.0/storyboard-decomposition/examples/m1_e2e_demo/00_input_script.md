# 《当霸道总裁拥有媒体素养 EP01》分镜头脚本

> **源文件**：`副本当霸道总裁拥有媒体素养 EP01(3).docx`
> **总时长**：约 60 秒
> **风格**：古早霸总风 → 反转普法/科普风
> **场景数**：5 大场 / 36 个镜头
> **制作格式**：横屏 16:9（建议）/ 9:16 竖屏亦可

---

## 0. 拍摄总则

| 项目 | 规范 |
|---|---|
| **画面风格** | 前半段：低饱和、强反差、慢动作、夸张仰拍（霸总美学）<br>后半段：自然光、真实色调、平稳镜头（去滤镜感） |
| **声音设计** | 两次"刹车声/唱片划痕"作为风格切换 cue；BGM 在转折点戛然而止 |
| **运镜语言** | 反转前：跟拍、推拉、慢动作<br>反转后：固定机位、对话式平视 |
| **特殊处理** | 结尾"普法先锋"光环 = 后期叠加；慢动作饮料 = 升格拍摄 120fps |

## 七、关键制作备忘

### 🎯 核心镜头（必拍精）
- **镜头 3.5 / 5.7**：两段刘一鸣长台词，**全片灵魂**。建议单机长镜头 + 多机位备份，演员台词功底需扎实。
- **镜头 6.4**："普法先锋"光环特效，是记忆锚点，**后期必须做扎实**。
- **镜头 1.5 / 2.9**：小仇的戏剧化表演是喜剧核心，**演员需要放得开**。

### ⚠️ 风险镜头（需打磨）
- **镜头 4.5**：刘一鸣"若有所思"的镜头容易被误读为"霸总上头"，建议：
  - 服装道具加强"工作属性"（手持文件夹/手机）
  - 镜头后续接 0.3 秒刘一鸣低头看工作手机，自然化解暧昧
- **镜头 1.1**：张园取景需提前报备，避免商业拍摄纠纷。
- **BGM**：不要直接套用某知名霸总剧 BGM，避免版权风险。

### 🎬 拍摄顺序建议
1. 先拍场景三、五（室内对话戏，演员状态最易控）
2. 再拍场景四（外景过渡戏，光线可控）
3. 最后拍场景一、二（外景 + 升格慢动作，对设备/天气要求高）

### 📋 演员要求
| 角色 | 关键词 | 试戏要点 |
|---|---|---|
| 刘一鸣 | 收放自如 | 既能"端着霸总架子"又能"瞬间理性冷静" |
| 小仇 | 戏精本精 | 表情管理 + 台词爆发力，**喜剧节奏感**是核心 |
| 骆骆 | 自然清新 | 不需演技炸裂，**真实可爱**即可 |

- **竖屏 9:16**：抖音、快手、小红书（需重新构图，把对话镜头拉紧）
