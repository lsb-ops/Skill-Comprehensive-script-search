## 三、工作流回放

### 步骤 1：剧本解析（M1）
输入：`剧本2.0/第五幕_收束普法升华/` 等场景文件

输出：场景列表 + 人物清单 + 5幕结构 + 剧本原词 + 动作清单

参考：`M1_剧本解析.md` 模块

### 步骤 2：分镜设计（M2）
输入：M1 输出 + `styles/STYLE_古早霸总.md`

输出：
- 分镜大纲（11 镜）
- 场景主控档案（2 个）
- 衔接表
- CUE 清单（3 个）
- 色温曲线

参考：`00_全剧架构.md` 和 `00_场景主控_*.txt`

### 步骤 3：提示词生成（M3）
输入：M2 输出

输出：11 个 `镜XXX_xxx.txt` 文件，每个 ≤1800 字（v3.7 治本）

参考：`T1_单镜提示词.md` 模板

### 步骤 4：衔接校验（M4）
输入：所有 txt 文件

输出：校验报告（issue 清单 + 修复建议）

参考：`00_衔接校验报告.md`

## 四、关键学习点

### 1. CUE 设计（3 个）
- **CUE①**：镜 004 开篇风格切换（配乐 + 摘墨镜）
- **CUE②**：镜 009 中段教化升级（刹车声 + 0.3 秒静默 + 冷化）
- **CUE③**：镜 011 升华（色温跳变 + 逆光剪影 + Halation）

### 2. 色温曲线
```
4500K → 5000K → 4500K → 4000K → 5500K
建制   触发    反转一   反转二   收束
```

### 3. 道具轨迹
- **皮鞋**：镜 001-002 出现，镜 010-011 强化
- **墨镜**：镜 004 戴上，镜 005-009 戴，镜 010-011 收于西装内袋
- **名片**：镜 007 关键转折

### 4. 人物弧光
- **刘一鸣**：冷漠霸总 → 教化者（普法先锋）
- **小仇**：嚣张 → 瑟瑟发抖（被教化）
- **骆骆**：飞快逃离（拒绝参与）
