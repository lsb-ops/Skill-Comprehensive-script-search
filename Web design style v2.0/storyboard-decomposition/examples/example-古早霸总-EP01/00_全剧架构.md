# 00 — 全剧架构（M2 输出）

> 古早霸总剧《当霸道总裁拥有媒体素养》EP01
> 总时长 80 秒，11 镜

---

## 一、5 幕结构

### 第 1 幕：建制（镜 001-002）
- **时长**：约 13 秒
- **场景**：张园 → 弄堂
- **内容**：刘一鸣走入张园，切换到弄堂与路人相撞
- **情绪**：复古高级感 → 自然

### 第 2 幕：触发（镜 003-004）
- **时长**：约 13 秒
- **场景**：弄堂
- **内容**：骆骆道歉、小仇咆哮、刘一鸣摘墨镜
- **情绪**：日常 → 戏剧张力
- **CUE**：镜 004 包含 CUE① 开篇风格切换

### 第 3 幕：反转一（镜 005-006）
- **时长**：约 13 秒
- **场景**：弄堂
- **内容**：刘一鸣长台词讲常识，小仇呆若木鸡
- **情绪**：戏剧张力 → 教化

### 第 4 幕：反转二（镜 007-010）
- **时长**：约 27 秒
- **场景**：弄堂 → 弄堂内
- **内容**：温情递名片、小仇邪魅、总裁普法长台词、小仇被教化、刘一鸣走开
- **情绪**：教化 → 升华
- **CUE**：镜 009 包含 CUE② 中段教化升级

### 第 5 幕：收束（镜 011）
- **时长**：约 8 秒
- **场景**：弄堂尽头
- **内容**：刘一鸣背影升华
- **情绪**：升华
- **CUE**：镜 011 包含 CUE③ 升华

## 二、CUE 清单

### CUE①：开篇风格切换
- **位置**：镜 004（0-3 秒）
- **类型**：配乐 + 摘墨镜 + 色温跳变
- **参数**：配乐渐起，刘一鸣摘墨镜
- **视觉效果**：从日常到戏剧张力
- **配乐**：钢琴/弦乐渐起
- **色温**：4500K → 5000K

### CUE②：中段教化升级
- **位置**：镜 009（0-3.3 秒）
- **类型**：关键音效 + 静默
- **参数**：刹车声 + 0.3 秒静默
- **视觉效果**：从日常到严肃教化
- **配乐**：静默后弦乐爆发
- **色温**：5000K → 4000K

### CUE③：升华
- **位置**：镜 011（0-2 秒）
- **类型**：色温跳变 + 逆光剪影
- **参数**：色温 4000K → 5500K 跳变
- **视觉效果**：从教化到胜利（普法先锋光环）
- **配乐**：高峰 + 内心 OS
- **光比**：1:∞（剪影）
- **Halation**：开
