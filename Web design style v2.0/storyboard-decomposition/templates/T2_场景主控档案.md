# T2 — 场景主控档案模板

> 用于定义一个场景的统一定义
> 同一场景的所有镜共享这份档案
> 字数：≤1800 字（v3.7 治本）
> 版本：v1.1（2026-06-17）—— 新增 v3.4 治本章节
> - 单镜提示词引用本档案时**不重复描述视觉信息**（年龄/服装/发型/表情/道具外观/场景细节），仅引用资产指针

---

## 字段说明

### 基本信息

| 字段 | 说明 | 示例 |
|---|---|---|
| 场景编号 | 格式 S0X | S01 |
| 场景名 | 简短场景名 | 弄堂 |
| 时间 | 日/夜/黄昏/黎明 | 日 |
| 地点 | 具体地点 | 老上海弄堂 |
| 覆盖镜号 | 该场景覆盖的镜号 | 镜 001-005，镜 009-011 |
| 主导情绪 | 情绪关键词 | 冷峻/教化 |
| 5幕位置 | 在5幕结构中的位置 | 第1幕 建制 |

### 视觉定义

| 字段 | 说明 | 示例 |
|---|---|---|
| 场景图 | 场景图编号 | 【场景_弄堂：】 |
| 背景元素 | 关键背景元素 | 青砖墙、石板路、晾衣杆 |
| 主导色温 | 色温范围 | 4500K-5000K |
| 主导光比 | 光比范围 | 1:2-1:3 |
| 主导色调 | 色调类型 | 中性青橙 |

### 镜头规格

| 字段 | 说明 | 示例 |
|---|---|---|
| 画幅 | 屏幕比例 | 9:16 |
| 机位 | 摄影机水平位置 | 轴线下方 |
| 高度 | 摄影机垂直角度 | 平视 |
| 帧率 | 帧率 | 24fps |

### 道具清单

格式：`<道具名>（<镜号范围>，<位置>，<延续关系>）`

示例：
- 皮鞋（镜 001-002，镜 010-011，刘一鸣脚上，v3.0 强化高光反射 60%）
- 墨镜（镜 004-010，刘一鸣脸上/收于西装内袋，金属框有自然轮廓）
- 名片（镜 007，小仇手中，关键道具）

### 人物清单

格式：`<人物名>（<角色定位>，<关键特征>）`

示例：
- 刘一鸣（P01，主角，黑色西装、Prada 领带、180cm+）
- 小仇（P02，对手，浅蓝西装、银框眼镜、凌乱头发）

### 场景主光

三点布光定义：

```
- 主光 Key Light：4000K 左上方 45 度，强度 80%
- 辅光 Fill Light：4500K 冷蓝，强度 30%
- 轮廓光 Back Light：5000K 中性偏暖，强度 80%
```

### 调色管线

| 字段 | 说明 | 示例 |
|---|---|---|
| 基础 LUT | 调色 LUT | Fuji Eterna |
| 胶片颗粒 | 胶片颗粒类型 | 35mm |
| Halation | 光晕开关 | 关（CUE 时刻开） |
| 特殊 LUT | 特殊 LUT | 升华 LUT（仅 CUE③） |

### 风格参考

3-4 部参考电影，每部说明参考点：

```
- 教父：黑帮美学、逆光剪影
- 天堂电影院：升华结尾、童年回忆
- 功夫：教化者温度、如来神掌光环
```

### 物理规则

场景内所有镜共享的物理规则：

```
- 走路速度：1.5 米/每秒真实行人速度
- 皮鞋踏步声：真实石板路触感
- 风速：1-2 米/秒（轻风）
- 衣料：自然褶皱、自然垂坠感
```

### 衔接规则

- 场景内：场景内镜的衔接方式（如同位置续拍）
- 场景外：场景外镜的衔接方式（如空镜过渡）

### 注意事项

本场景的特别注意事项：

```
- 同场景多镜建议合并为一次生成
- 镜头拍摄建议一次性拍完整场戏
- CUE② 关键音效出现位置需精确
```

```markdown
## 场景主控档案

### 场景 S01：弄堂

#### 基本信息
- 场景编号：S01
- 场景名：弄堂
- 时间：日
- 地点：老上海弄堂
- 覆盖镜号：镜 001-005，镜 009-011
- 主导情绪：冷峻/教化
- 5幕位置：第1幕 建制 + 第5幕 收束

#### 视觉定义
- 场景图：【场景_弄堂：】
- 背景元素：青砖墙、石板路、晾衣杆、藤蔓、老式路灯
- 主导色温：4500K-5500K（CUE 时刻跳变）
- 主导光比：1:2-1:∞（CUE③ 升华剪影）
- 主导色调：中性青橙（CUE③ 升华暖金）

#### 镜头规格
- 画幅：9:16
- 机位：固定在轴线下方
- 高度：平视
- 帧率：24fps

#### 道具清单
- 皮鞋（镜 001-002，镜 010-011，刘一鸣脚上，v3.0 强化高光反射 60%）
- 墨镜（镜 004-010，刘一鸣脸上/收于西装内袋，金属框有自然轮廓）
- 名片（镜 007，小仇手中，关键道具）

#### 人物清单
- 刘一鸣（P01，主角，黑色西装、Prada 黑色领带、深蓝衬衫、180cm+、黑短发）
- 小仇（P02，对手，浅蓝西装、银框眼镜、白色印花 T 恤、175cm、凌乱短发）

#### 场景主光
- 主光 Key Light：4000K 弄堂天光柔光 左上方 45 度，强度 80-90%
- 辅光 Fill Light：4500K 冷蓝，强度 30%
- 轮廓光 Back Light：5000K 中性偏暖，强度 80%

#### 调色管线
- 基础 LUT：Fuji Eterna
- 胶片颗粒：35mm
- Halation：关（CUE③ 升华时开）
- 特殊 LUT：升华 LUT（仅 CUE③）+ 暖金 LUT

#### 风格参考
- 教父：黑帮美学、逆光剪影
- 阿拉伯的劳伦斯：逆光剪影
- 天堂电影院：升华结尾、童年回忆
- 功夫：教化者温度、如来神掌光环

#### 物理规则
- 走路速度：1.5 米/每秒真实行人速度
- 皮鞋踏步声：真实石板路触感
- 风速：1-2 米/秒（轻风）
- 衣料：自然褶皱、自然垂坠感
- 瑟瑟发抖频率：每秒 4-6 次肌肉颤动

#### 衔接规则
- 场景内：场景内镜建议连续生成（同位置续拍）
- 场景外：用 LS 远景过渡，或色温跳变暗示时间变化

#### 注意事项
- 同场景多镜建议合并为一次生成
- 镜头拍摄建议一次性拍完整场戏
- CUE② 关键音效（刹车声 + 0.3 秒静默）出现位置需精确
- CUE③ 升华色温跳变 + 逆光剪影 + Halation 三件套必须到位
- 墨镜位置跨镜连贯（镜 004-009 戴，镜 010-011 收于西装内袋）
```

## 使用流程

### 第1步：场景识别
从 M1 输出的场景列表中识别每个场景。

### 第2步：场景主控档案生成
对每个场景，套用 T2 模板填充所有字段。

### 第3步：跨场景一致性检查
- 同一场景的所有镜必须共享这份档案
- 不同场景之间要保持色温/光比的连贯性

### 第4步：场景主控档案作为 M2/M3 的输入
- M2（分镜设计）：用场景主控档案的视觉定义设计分镜
- M3（提示词生成）：用场景主控档案填充各镜的相同字段

## 与单镜提示词的关系

| 字段 | 场景主控档案 | 单镜提示词 |
|---|---|---|
| 主导色温 | 给出范围 | 给出当前镜的具体色温 |
| 主导光比 | 给出范围 | 给出当前镜的具体光比 |
| 场景图 | 定义 | 引用 |
| 道具清单 | 列出所有道具 | 引用本镜出现的道具 |
| 人物清单 | 列出所有人物 | 引用本镜出现的人物 |
| 场景主光 | 定义三点布光 | 引用 + 标注延续/变化 |
| 调色管线 | 定义基础 LUT | 引用 + 标注变化 |
| 风格参考 | 列出所有参考 | 引用 3-4 部 |

**关键**：单镜提示词引用场景主控档案，不重复定义。

## v3.4 治本：场景主控档案与单镜提示词的协同（NEW）

### 核心原则

**场景主控档案 = 视觉信息唯一权威来源**
**单镜提示词 = 引用指针 + 镜内独特信息**

场景主控档案已包含完整的视觉定义（人物外貌/服装/道具外观/场景细节），单镜提示词引用即可，**不得重复描述**。

### 引用模板

**单镜提示词的"人物资产档案"段（v3.4 格式）**：
```markdown
人物资产档案（v3.4 治本·参考图已承载信息则提示词不重复）
【本镜人物外貌/服装/发型/表情 全部由参考图承载，提示词不重复描述】
【参考_人物_刘一鸣：03_人物档案/人物_刘一鸣.png（关键特征·参考图承载）】
【参考_人物_小仇：03_人物档案/人物_小仇.png（关键特征·参考图承载）】
【参考_场景_张园外景：02_场景主控/场景_张园外景.png（关键特征·参考图承载）】
【合规保留】严禁字幕文字：xxx（图片无法承载的合规/禁忌信息）
```

### 单镜可保留的"镜内独特信息"

✅ **可保留**（本镜独有，场景主控档案无法承载）：
- 本镜特定状态（如"无墨镜·露脸·M11决策#005协调"）
- 本镜人物坐标位移（如"刘一鸣 MS 居中→骆骆 MS 偏右"）
- 本镜色温跳变（CUE 时刻）
- 本镜道具状态变化（如"皮鞋高光反射 60%→70%"）

❌ **禁止重复**（场景主控档案已承载）：
- 年龄/性别/身高/体型
- 服装颜色/品牌/材质
- 发型/发色
- 表情描述
- 道具外观（颜色/材质/形状）
- 场景细节（建筑/装饰/天气）
