# KB12 — 声音参考治本（v3.7）

> **目标**：在 AI 视频分镜提示词中加入"声音参考"模块，让 AI 视频生成时能精确还原角色音色
> **方法**：4 维度设计 + 3 角色基线 + 8 镜变体 + 视觉-声音资产对齐
> **适用**：所有含台词的镜（无对白镜不加）
> **基于**：EP01 11 镜实测（8 镜加声音 / 3 镜无对白）

---

## 一、为什么需要声音参考？

### 1.1 问题（v3.6 治本前）

- AI 视频 API（即梦/可灵）生成的视频中，**角色音色是随机的**
- 同一角色在不同镜中可能出现**完全不同的音色**（男/女/老/少/方言）
- 即使台词相同，刘一鸣在镜 005 和镜 009 的声音可能天差地别
- **结果**：26 镜 EP01 中"刘一鸣"听起来像 5 个不同的人

### 1.2 解决方案（v3.7 治本）

- 在资产引用段加入 `【声音_角色名：】` 模块
- 用 4 维度（音色/语速/共鸣/风格）精确描述声音特征
- AI 视频 API 据此选择/微调音色
- **跨镜一致性**：同角色在不同镜中保持音色基线，仅风格微调

## 二、声音参考格式标准

### 2.1 格式

```
【声音_角色名：音色·语速X字/分·共鸣位置·风格特征】
```

**示例**：

```
【声音_刘一鸣：男中音·沉稳·语速280字/分·胸腔共鸣·理性克制·逻辑强教化者】
【声音_小仇：男中音偏高·夸张尖锐·语速350字/分·口腔共鸣·戏剧化谄媚】
【声音_骆骆：女高音·细软发颤·语速180字/分·头腔共鸣·紧张惊慌】
```

### 2.2 4 维度设计

| 维度 | 取值范围 | 作用 |
|------|----------|------|
| **音色** | 男中音/男中音偏高/男中音低沉沙哑/女高音/女中音 | 性别 + 音高定位 |
| **语速** | X 字/分钟（150-350）| 节奏感，决定角色气质 |
| **共鸣位置** | 胸腔（沉稳）/ 口腔（戏精）/ 头腔（清澈）| 音色质感 |
| **风格特征** | 理性克制/邪魅神秘/紧张惊慌/温和克制/严肃权威/回味反思 | 情绪基线 |

### 2.3 格式约束

- 总长度 ≤ 60 字符（保持紧凑）
- 用 `·` 分隔各维度
- 最后一个维度可省略（如纯音色场景）
- 必须与 `【人物_角色名：】` 配对使用

## 六、M3 生成流程（声音引用自动化）

### 6.1 自动注入逻辑

```python
def inject_voice_reference(mirror_id, character_lib):
    """M3 生成时，自动从 voice.json 注入声音参考"""
    
    # 1. 加载角色声音档案
    voice = character_lib.load(f"voice_{character_id}.json")
    
    # 2. 查找本镜台词
    if not mirror.has_dialogue():
        return ""  # 无对白镜不注入
    
    # 3. 生成【声音_xxx：】行
    ep_data = voice.get_ep_appearance(mirror.ep_id, mirror.id)
    voice_line = f"【声音_{character_name}：{ep_data.timbre}·语速{ep_data.speed_wpm}字/分·{ep_data.resonance}·{ep_data.style}】"
    
    # 4. 插入到【人物_xxx：】之后
    asset_section = inject_after_character_line(
        asset_section, character_id, voice_line
    )
    
    return asset_section
```

### 6.2 触发判断

| 判断 | 代码 | 结果 |
|------|------|------|
| 该镜台词列表非空 | `len(dialogue) > 0` | ✅ 注入 |
| 台词包含画外音/内心 OS | `dialogue_type in ['画外音', '内心OS']` | ✅ 注入 |
| 台词纯动作指示（如舞台指示）| `dialogue_type == '动作'` | ❌ 跳过 |

## 七、校验规则（M7 AI Verify 扩展）

### 7.1 新增校验维度

| 维度 | 满分 | 校验规则 |
|------|------|----------|
| **声音引用完整性** | 10 分 | 有台词镜必须有 `【声音_xxx：】`，缺一扣 5 分 |
| **声音 4 维度齐全** | 10 分 | 音色+语速+共鸣+风格 缺一扣 2.5 分 |
| **角色声音基线一致** | 10 分 | 同角色不同镜音色/共鸣/语速区间在基线范围内 |

### 7.2 阈值

- 总分 ≥ 25/30 = 通过
- 总分 < 25 = needs_redo

```python
def check_voice_reference(text, dialogue_section):
    """v3.7 新增：声音引用校验"""
    score = 0
    issues = []
    
    # 1. 完整性
    if dialogue_section and "声音_" not in text:
        issues.append("有台词但缺声音引用")
    elif not dialogue_section and "声音_" in text:
        issues.append("无对白但有声音引用（应删除）")
    else:
        score += 10
    
    # 2. 4 维度齐全
    voice_lines = re.findall(r"声音_[^：]+：([^】]+)", text)
    for line in voice_lines:
        dims = line.split("·")
        if len(dims) >= 3:
            score += 10/len(voice_lines)
        else:
            issues.append(f"声音描述少于3维度：{line}")
    
    return score, issues
```

```python
def inherit_voice_reference(ep02_mirror, ep01_voice_lib):
```

## 十一、常见错误速查

| 错误 | 后果 | 修复 |
|------|------|------|
| 声音引用与人物引用不匹配 | 角色对不上 | 确保 `【声音_角色：】` 紧跟 `【人物_角色：】` |
| 4 维度少于 3 个 | AI 抓不到重点 | 必须包含音色+语速+共鸣 |
| 语速单位错误 | AI 生成混乱 | 必须用"X 字/分"格式 |
| 共鸣位置不明确 | 音色飘忽 | 必须用"胸腔/口腔/头腔"标准词 |
| 风格描述抽象（如"好听"）| 不可执行 | 必须用具体特征（"理性克制"/"邪魅神秘"）|
| 无对白镜加声音引用 | 冗余 + 字数超标 | 删除 |
| 声音引用超出 60 字符 | 字数超标 | 精简描述，保留核心 3 维度 |
