# KB03 — 光影色温速查

> 三点布光 + 色温曲线 + 光比 + 色调

---

## 一、三点布光（Three-Point Lighting）

### 主光 Key Light
- **作用**：定义主体形态、主要光源
- **位置**：45 度角斜上方（左上或右上）
- **强度**：100%（基准）
- **色温**：决定整体情绪（暖/冷/中性）
- **硬度**：硬光有清晰阴影，柔光阴影渐变

### 辅光 Fill Light
- **作用**：补阴影、控制对比度
- **位置**：与主光相对一侧（90 度到 180 度）
- **强度**：30%-50%（主光的 1/2 到 1/3）
- **色温**：通常与主光相近或偏冷
- **目的**：让阴影不死黑，保留细节

### 轮廓光 Back Light / Rim Light
- **作用**：勾轮廓、分离主体与背景
- **位置**：正后方或后上方
- **强度**：80%-100%（可强于主光制造剪影）
- **色温**：通常偏暖（金/橙）
- **目的**：让主角"发光"，增强立体感

## 二、色温曲线

### 基础色温对照

| 色温 K | 视觉 | 情绪 | 适用场景 |
|---|---|---|---|
| 2700K | 烛光、夕落 | 亲密、温暖、回忆 | 暖场、情感戏 |
| 3200K | 钨丝灯 | 怀旧、温馨 | 室内灯、夜场 |
| 4000K | 晨昏、自然暖阴 | 冷静、过渡 | CUE 教化、科普 |
| 4500K | 晨光 | 清新、希望 | 开篇、白天 |
| 5000K | 日光、白天 | 中性、客观 | 日常叙事 |
| 5500K | 晴天正午 | 明亮、清晰 | CUE 升华、胜利 |
| 6500K | 阴天、电子 | 冷淡、孤独 | 悬疑、紧张 |
| 8000K+ | 雪景、阴影 | 寒冷、死亡 | 悲剧、困境 |

### 短剧典型色温曲线（5 幕结构）

```
建制（幕1）：4500K 清新 → 5000K 自然
触发（幕2）：5000K → 4500K 微妙降温
反转一（幕3）：4500K → 4000K 冷化
反转二（幕4）：4000K → 4500K 缓和 → 4000K 加深教化
收束（幕5）：4000K → 5500K CUE③升华（色温跳变）
```

### CUE 时刻的色温跳变

**CUE① 开篇风格切换**：
- 2700K（暖场） → 4500K（建立正常基调）
- 视觉：从亲密到清晰

**CUE② 中段教化升级**：
- 5000K → 4000K（冷静科普美学）
- 视觉：从日常到严肃

**CUE③ 升华**：
- 4000K → 5500K（普法先锋的光环美学）
- 视觉：从教化到胜利
- 必须显式标注"色温跳变" + 真实物理过渡

## 三、光比（Lighting Ratio）

### 光比定义
- 主光亮度 : 辅光亮度
- 例如 1:2 表示主光是辅光的 2 倍

### 常见光比

| 光比 | 视觉 | 情绪 | 适用 |
|---|---|---|---|
| 1:1 | 平光、阴影少 | 客观、记录 | 综艺、采访 |
| 1:2 | 柔光、轻微阴影 | 自然、温和 | 日常叙事 |
| 1:3 | 中等对比 | 戏剧、聚焦 | 对话戏、心理戏 |
| 1:4 | 强对比、明显阴影 | 紧张、悬疑 | 悬疑剧、惊悚 |
| 1:8 | 极强对比、剪影 | 戏剧张力 | CUE 时刻、升华 |
| 1:∞ | 纯剪影、纯逆光 | 神化、升华 | 结尾升华、背影 |

### 短剧典型光比

- **日常对话**：1:2 柔光
- **心理戏**：1:3 中等对比
- **CUE 高潮**：1:4 强对比
- **升华剪影**：1:∞ 纯逆光

## 四、色调（Color Tone）

### 三大基础色调

#### 中性青橙（Teal & Orange）
- 高光：青（7000K-8000K）
- 阴影：橙（3000K-3500K）
- 视觉：电影感、好莱坞
- 适用：90% 商业片、现代剧
- LUT：Fuji Eterna、Teal & Orange LUT

#### 暖金（Warm Gold）
- 全画面：5500K-6500K 偏暖
- 视觉：神化、升华、神圣
- 适用：CUE③ 升华、胜利时刻
- LUT：Warm Gold LUT

#### 冷蓝（Cool Blue）
- 全画面：6500K-8000K
- 视觉：孤独、悬疑、冷静
- 适用：悲剧、心理、悬疑
- LUT：Bleach Bypass、Cold Blue

### 短剧典型色调

- **建制**：中性青橙
- **触发**：中性青橙 + 微妙暖化
- **反转一**：中性青橙 + 冷化
- **反转二**：中性青橙 + 严肃
- **升华**：暖金（普法先锋的光环美学）

## 五、调色管线（Color Grading Pipeline）

### LUT 选择

| LUT | 风格 | 适用 |
|---|---|---|
| Fuji Eterna | 电影感、柔和肤色 | 通用默认 |
| Kodak 2383 | 复古、暖调 | 怀旧、90年代 |
| Teal & Orange | 现代商业片 | 现代剧、霸总 |
| Bleach Bypass | 高对比、灰调 | 悬疑、战争 |
| Warm Gold | 升华、神化 | 升华时刻 |
| Halation | 光晕、梦幻 | 升华、梦境 |

### 胶片颗粒（Film Grain）

- 35mm 胶片颗粒：温暖、有机感
- 16mm 胶片颗粒：粗犷、纪实
- 无颗粒：干净、数字感

### Halation（光晕）

- 强光周围红/橙光晕
- 视觉：电影感、梦幻、升华
- 适用：CUE 升华、梦境
- 不适用：日常叙事（容易廉价）

## 七、光影叙事规律

### 暖→冷 = 危机/恶化
- 4500K → 4000K：进入问题
- 5000K → 6500K：危机深化

### 冷→暖 = 救赎/胜利
- 4000K → 5500K：教化成功（CUE③）
- 6500K → 4500K：回到日常

### 暗→亮 = 揭示/觉醒
- 1:4 → 1:2：心理缓解
- 1:∞ → 1:2：从剪影到正常

### 亮→暗 = 危机/死亡
- 1:2 → 1:4：进入阴影
- 1:2 → 1:∞：剪影化死亡
