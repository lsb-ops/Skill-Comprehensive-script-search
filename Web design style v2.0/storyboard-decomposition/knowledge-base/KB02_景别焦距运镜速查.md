## 二、景别详解

### ECU（Extreme Close-Up 极致特写）
- 取景范围：仅一个器官（眼/嘴/手）
- 焦距：85-100mm
- 视觉心理：侵入感、强烈情绪、心理冲击
- 适用场景：CUE 时刻、高潮点、道具细节
- 占画面纵向：80-100%

### CU（Close-Up 特写）
- 取景范围：头部+肩部
- 焦距：50-85mm
- 视觉心理：亲密感、情绪捕捉
- 适用场景：对白、表情戏、戏剧反应
- 占画面纵向：60-80%

### MCU（Medium Close-Up 中近景）
- 取景范围：胸部以上
- 焦距：50-75mm
- 视觉心理：对话、互动
- 适用场景：双人对话、情绪对峙
- 占画面纵向：50-65%

### MS（Medium Shot 中景）
- 取景范围：腰部以上
- 焦距：35-50mm
- 视觉心理：动作、姿态
- 适用场景：动作戏、互动戏、走路
- 占画面纵向：40-60%

### MLS（Medium Long Shot 中全景）
- 取景范围：膝盖以上
- 焦距：35mm
- 视觉心理：人物+环境关系
- 适用场景：走位、展示空间
- 占画面纵向：30-50%

### FS（Full Shot 全景）
- 取景范围：全身
- 焦距：35mm
- 视觉心理：人物+环境
- 适用场景：登场、定点动作
- 占画面纵向：80-95%

### LS（Long Shot 远景）
- 取景范围：人物小，环境大
- 焦距：24-35mm
- 视觉心理：环境为主、人物渺小
- 适用场景：场景建立、氛围渲染
- 占画面纵向：10-30%

### ELS（Extreme Long Shot 大远景）
- 取景范围：人物几乎不可见
- 焦距：14-24mm
- 视觉心理：史诗感、孤独
- 适用场景：开篇、结尾升华、城市/自然全景
- 占画面纵向：<10%

## 三、焦距详解

### 广角（14-35mm）
- 透视：夸张、变形
- 景深：深（背景清晰）
- 视觉心理：空间感、不稳定、动作张力
- 适用：动作戏、环境展示、心理失衡

### 标准（35-50mm）
- 透视：接近人眼
- 景深：中
- 视觉心理：自然、客观
- 适用：MS 中景、对话戏、日常叙事

### 中长焦（50-85mm）
- 透视：轻微压缩
- 景深：浅（背景虚化）
- 视觉心理：亲密、聚焦
- 适用：CU/MCU 特写、情绪戏

### 长焦（85-200mm）
- 透视：压缩明显
- 景深：极浅
- 视觉心理：窥视、隔离、心理压迫
- 适用：ECU 极致特写、远距离抓拍

## 四、运镜详解

### 固定机位
- 视觉心理：稳定、客观、凝视
- 适用：对白、CU 特写、静态构图

### 推（Dolly In / Push In）
- 速度：0.2-1.0 米/秒
- 视觉心理：逼近、压力、情绪升级
- 物理：从 MS 推到 CU，每秒 0.3 米
- 适用：CUE 时刻、高潮逼近、心理压力

### 拉（Dolly Out / Pull Back）
- 速度：0.2-1.0 米/秒
- 视觉心理：远离、释放、揭示
- 物理：从 CU 拉到 FS，每秒 0.3 米
- 适用：场景建立、人物孤立、收束

### 横移（Truck / Tracking）
- 速度：0.5-1.5 米/秒（行人速度 1.5）
- 视觉心理：跟随、平行、记录
- 适用：走路、追逐、平行叙事

### 摇（Pan）
- 速度：5-30 度/秒
- 视觉心理：揭示、跟随视线
- 适用：揭示空间、跟随人物转头

### 跟（Follow）
- 视觉心理：主观、亲密
- 适用：POV 视角、人物背后

### 升降（Pedestal / Crane）
- 视觉心理：权力、神性、渺小
- 适用：戏剧性开场、收束

### Zoom（变焦）
- 与 Dolly 区别：变焦改变焦距，Dolly 改变机位距离
- Zoom 视觉心理：突然、压迫、梦境
- Dolly 视觉心理：物理、真实、空间
- 短剧慎用 Zoom（容易廉价）

### 缓推/缓拉（Slow Dolly）
- 速度：0.2-0.5 米/秒
- 视觉心理：微妙、暧昧、渐进
- 适用：情绪过渡、暧昧、思考

## 五、机位 + 高度

### 机位
- 左机位：从人物左侧拍摄
- 右机位：从人物右侧拍摄
- 居中：正面或正后方
- **180 度轴线规则**（见 KB04）：一旦设定机位，所有后续镜头在同一侧

### 高度
- 平视（Eye Level）：最常用、客观
- 俯视（High Angle）：人物渺小、脆弱、审视
- 仰视（Low Angle）：人物高大、权力、神化
- 荷兰角（Dutch Angle）：倾斜 15-30 度，不安、心理失衡
- 鸟瞰（Bird's Eye）：垂直俯视，全知视角

---

## 六、视角 + 视点

### 视角
- 客观视角（Objective）：观众视角，第三方观察
- 主观视角（POV）：人物视角，看到人物看到的
- 第一人称：常用于惊悚、悬疑、亲密

### 视点（视线方向）
- **轴线规则核心**：演员 A 视线朝左，演员 B 视线朝右（建立左右空间）
- 一旦建立，全剧保持一致
- 反打镜头：180 度切换视点

---

## 七、构图

### 中心构图
- 视觉心理：稳定、权威、孤独
- 适用：肖像、孤独角色

### 三分构图
- 视觉心理：自然、电影感
- 适用：70% 场景

### 对称构图
- 视觉心理：庄严、对比、心理对峙
- 适用：对峙、仪式感

### 对角线构图
- 视觉心理：动态、紧张、不稳定
- 适用：动作、追逐、悬疑

### 留白构图
- 视觉心理：孤独、渺小、空间感
- 适用：情感高潮、背影、收束

### 前景构图
- 视觉心理：层次、窥视
- 适用：双人对峙、隐藏情绪

## 八、景深

### 浅景深（f/1.4-2.8）
- 视觉心理：聚焦、亲密、梦境
- 适用：CU 特写、情绪戏
- 长焦镜头（85mm+）自动浅景深

### 中景深（f/4-5.6）
- 视觉心理：自然、平衡
- 适用：MS 中景、对话

### 深景深（f/8-16）
- 视觉心理：环境、空间、记录
- 适用：LS 远景、动作戏

---

## 九、帧率

| 帧率 | 视觉 | 适用 |
|---|---|---|
| 24 fps | 电影感、温暖、轻微拖影 | 剧情片、文艺 |
| 25 fps | PAL 制式，接近 24 | 欧洲剧 |
| 30 fps | 电视感、清晰 | 综艺、日常 |
| 60 fps | 流畅、HFR | 动作、慢动作 |

**短剧推荐**：24 fps（电影感基础）

## 十、常见组合配方

### 对话戏
```
景别：MCU/MS 双人正反打
焦距：50mm
运镜：固定
构图：三分
景深：浅（f/2.8）
```

### 动作戏
```
景别：MS/MLS
焦距：24-35mm
运镜：跟拍/横移 1.5 米/秒
构图：对角线
景深：中（f/4-5.6）
```

### CUE 高潮
```
景别：CU → ECU 推
焦距：85mm
运镜：缓推 0.3 米/秒
构图：中心
景深：极浅（f/1.4）
```

### 背影收束
```
景别：MS → LS
焦距：35mm → 24mm
运镜：缓拉 0.3 米/秒
构图：留白中心
景深：中
```

### 教化者温度
```
景别：MCU 居中
焦距：50mm
运镜：固定
构图：三分
景深：浅
细节：眉头紧锁但嘴角微动
```

## 十一、检验清单

- [ ] 景别变化是否合理（不要频繁跳景别）
- [ ] 焦距是否匹配景别
- [ ] 运镜速度是否有具体数值
- [ ] 机位是否遵守 180 度轴线
- [ ] 构图是否有主体位置
- [ ] 景深是否匹配情绪
- [ ] 帧率是否统一（全剧 24fps）
