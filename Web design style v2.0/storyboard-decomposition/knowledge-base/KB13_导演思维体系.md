## 二、10 维度详解

### 维度 1：戏剧意图（Dramatic Intent）

**定义**：这一镜想让观众**感受什么**。

**为什么缺失**：现有 skill 只关注"拍什么"（动作/对白），不关注"为什么拍"（观众感受）。

**示例**：

| 镜号 | 戏剧意图（v3.8 强制） |
|---|---|
| 镜 001 | 让观众感受**霸总的压迫感**（建立人物权威） |
| 镜 002 | 让观众感受**意外的紧张**（打破预期，蓄势） |
| 镜 004 | 让观众感受**戏剧反转的荒诞**（CUE 跳变瞬间） |
| 镜 009 | 让观众感受**教化者的力量**（台词权威感） |
| 镜 011 | 让观众感受**升华的孤高**（背影美学） |

**模板**：
```markdown
【戏剧意图】让观众感受 [情绪关键词]。
              触发条件：[剧本中的关键动作/对白]。
              期待反应：[观众应该有的认知或情感反应]。
```

---

### 维度 2：情感弧线（Emotional Arc）

**定义**：角色在这一镜中**情感起点 → 终点**的轨迹。

**为什么缺失**：现有 skill 只记录"角色做了什么"，不记录"角色经历了什么心理变化"。

**示例**：

| 镜号 | 角色 | 情感起点 | 情感终点 | 弧线类型 |
|---|---|---|---|---|
| 镜 001 | 刘一鸣 | 冷漠 | 冷漠（稳定） | 零弧 |
| 镜 002 | 骆骆 | 无辜 | 惊慌 | 突变（外部事件触发） |
| 镜 005 | 刘一鸣 | 无语 | 教化者（理性权威） | 升弧（情绪升级） |
| 镜 009 | 刘一鸣 | 教化者 | 普法者（更严肃） | 升弧（第二次升级） |
| 镜 011 | 刘一鸣 | 教化者 | 升华（回味反思） | 收弧（情绪归位） |

**模板**：
```markdown
【情感弧线】[角色名]：从 [起点状态] → [终点状态]（[弧线类型：突变/升弧/降弧/收弧/零弧]）。
              触发事件：[导致情感变化的剧本事件]。
              身体映射：[情感的身体化表现，如眉头紧蹙/呼吸急促]。
```

---

### 维度 3：潜台词（Subtext）

**定义**：角色**没说出但藏在台词/动作背后的话**。

**为什么缺失**：现有 skill 把对白当作"台词"处理，不解读"台词背后的意思"。

**示例**：

| 镜号 | 表面台词 | 潜台词 |
|---|---|---|
| 镜 005（刘一鸣） | "什么叫'全球仅此一件的绝版高定'？我又不是明星走红毯" | "别再用夸张叙事侮辱我的智商" |
| 镜 005（刘一鸣） | "动不动就'打十辈子工赔不起'，这不仅涉嫌人格侮辱" | "你的价值观有问题" |
| 镜 009（刘一鸣） | "公民的个人信息受法律保护！未经他人同意，非法收集" | "你刚才想做的事是违法的，知不知道？" |
| 镜 009（刘一鸣） | "你信不信警察十分钟后就来把你带走？" | "我是认真的，收手吧" |

**模板**：
```markdown
【潜台词】[角色名]说"[表面台词]"。
              潜台词是："[没说但藏在背后的意思]"。
              演员提示：[如何通过表情/语气/身体语言传达潜台词]。
```

### 维度 4：视线引导（Eye-line Guidance）

**定义**：导演通过画面**引导观众看哪里**（焦点/次焦点/视线方向）。

**为什么缺失**：现有 KB04 站位轴线是"技术约束"（不能跳轴），不是"视线引导"（观众应该看哪里）。

**示例**：

| 镜号 | 主焦点 | 次焦点 | 视线方向 |
|---|---|---|---|
| 镜 002（相撞段） | 骆骆 CU 表情 | 刘一鸣前景模糊袖口 | 观众看骆骆反应（被泼的人） |
| 镜 003（小仇咆哮） | 小仇 MCU 咬牙切齿 | 骆骆 CU 前景模糊身体后缩 | 观众看小仇（攻击者） |
| 镜 005（刘一鸣教化） | 刘一鸣 MS 居中 | 小仇/骆骆前景模糊 | 观众看刘一鸣（教化者） |
| 镜 011（背影） | 刘一鸣背影剪影 | 摩天楼反光 | 观众看背影（升华美学） |

**模板**：
```markdown
【视线引导】主焦点：[本镜画面核心，观众第一眼看的]。
              次焦点：[次要关注点，引导视线转移]。
              视线方向：[角色/观众应该看哪个方向]。
              焦点转移：[0-X 秒看哪里 → Y-Z 秒看哪里]。
```

### 维度 5：节奏设计（Pacing Design）

**定义**：这一镜的**节奏类型**（紧张/舒缓/爆发/留白）。

**为什么缺失**：现有 skill 的 CUE 设计只关注"戏剧断点"，不关注"每镜节奏"。

**节奏类型 5 种**：

| 类型 | 特征 | 适用场景 | 镜头语言 |
|---|---|---|---|
| **紧凑** | 快切/多动作/紧张 | 冲突/追逐/高潮 | 短镜/MS+CU 快切/手持微抖 |
| **舒缓** | 长镜/慢推/自然 | 抒情/过渡/日常 | 长镜/MS 缓推/固定机位 |
| **爆发** | 突然动作/强情绪 | 戏剧反转/教化/高潮 | MCU→CU 急推/音效冲击 |
| **留白** | 空镜/静默/慢动作 | 沉思/情感过渡 | LS 远景/慢动作 0.5x/环境音 |
| **零节奏** | 静止/对白/思考 | 关键对白/独白 | CU 固定/微推 |

**示例**：

| 镜号 | 节奏类型 | 镜头语言映射 |
|---|---|---|
| 镜 002（相撞） | 紧凑→慢动作（3-5秒） | MS 快切 + 0.5x 慢动作 |
| 镜 003（小仇咆哮） | 爆发 | MCU→CU 急推 + 咬牙切齿 |
| 镜 005（教化长台词） | 舒缓 | MS 缓推 0.3 m/s + 自然音 |
| 镜 009（普法长台词） | 爆发 | MS 缓推 + 戴墨镜动作 + 严肃权威 |
| 镜 011（背影升华） | 留白 | MS 缓Dolly In/Out + 配乐高峰 |

**模板**：
```markdown
【节奏设计】类型：[紧凑/舒缓/爆发/留白/零节奏]。
              节奏曲线：[本镜节奏如何变化，如"0-2秒紧凑→3-5秒留白"]。
              镜头语言：[景别/运镜/音效如何映射节奏]。
```

### 维度 6：视觉隐喻（Visual Metaphor）

**定义**：用**画面元素**传达**抽象主题**。

**为什么缺失**：现有 skill 只关注"画面是什么"，不关注"画面代表什么"。

**示例**：

| 镜号 | 视觉元素 | 隐喻 |
|---|---|---|
| 镜 001 | 皮鞋踏步 + 张园石库门 | "权势的脚步声踏入旧上海" |
| 镜 002 | 弄堂相撞 + 白水泼洒 | "意外的纯净打破霸总滤镜" |
| 镜 004 | 摘墨镜 + 真实表情 | "撕下假面，回归真实" |
| 镜 011 | 背影剪影 + 摩天楼 | "个体在时代巨浪中的孤独与坚守" |

**模板**：
```markdown
【视觉隐喻】画面元素：[具体的视觉符号]。
              隐喻主题：[它代表的抽象概念]。
              主题服务：[如何强化剧集主题]。
```

### 维度 7：反差与对比（Contrast & Comparison）

**定义**：本镜与**前后镜**的**对比设计**（景别/节奏/光影/情绪）。

**为什么缺失**：现有 M2 衔接表只关注"位置延续"，不关注"情绪/节奏对比"。

**对比维度 5 种**：

| 维度 | 对比示例 |
|---|---|
| 景别 | 前镜 LS 远景 → 本镜 CU 特写（紧凑对比） |
| 节奏 | 前镜舒缓 → 本镜爆发（节奏对比） |
| 光影 | 前镜暖光 → 本镜冷光（情绪对比） |
| 情绪 | 前镜冷漠 → 本镜慌张（情感对比） |
| 构图 | 前镜居中 → 本镜偏左（视觉张力） |

**示例**：

| 镜号 | 与前镜对比 | 设计意图 |
|---|---|---|
| 镜 004 | 镜003 MCU咆哮 → 镜004 MCU→CU 摘墨镜 | **节奏反差**（爆发→舒缓） |
| 镜 009 | 镜008 小仇邪魅 → 镜009 戴墨镜严肃 | **情绪反差**（邪魅→权威） |
| 镜 011 | 镜010 背影走开 → 镜011 升华背影 | **升华延续**（同方向强化） |

**模板**：
```markdown
【反差对比】与前镜：[前镜特征] → 本镜：[本镜特征]（[反差类型]）。
              与后镜：[后镜预告]（[反差/延续类型]）。
              设计意图：[为什么这样对比，服务什么叙事]。
```

### 维度 8：主题表达（Thematic Expression）

**定义**：这一镜**如何服务剧集核心主题**。

**为什么缺失**：现有 skill 把"主题"当作高层概念，不下沉到每镜。

**示例（霸总剧主题 = "媒介素养"）**：

| 镜号 | 主题服务方式 |
|---|---|
| 镜 002 | 用"夸张叙事碰撞日常"建立反差，引出主题 |
| 镜 005 | 用"教化台词"直接传达"媒介素养"概念 |
| 镜 009 | 用"普法台词"具体化"媒介素养"实践 |
| 镜 011 | 用"背影"升华主题（教化完成但孤独） |

**模板**：
```markdown
【主题表达】剧集主题：[剧集核心主题的一句话]。
              本镜服务：[如何用画面/对白/动作传达主题]。
              主题强度：[强/中/弱，本镜对主题的承载程度]。
```

### 维度 9：角色心理关系（Character Psychology）

**定义**：镜中**角色之间的心理权力关系**（主导/跟随/对抗/和解）。

**为什么缺失**：现有 KB04 站位是"物理位置"，不是"心理关系"。

**心理关系 4 类**：

| 类型 | 镜头语言 | 示例 |
|---|---|---|
| **主导** | A 居中 + B 模糊前景 | 刘一鸣教化（镜005/009） |
| **跟随** | B 在 A 后方/侧方 | 小仇紧随刘一鸣（镜001） |
| **对抗** | A B 居中对峙 | 小仇咆哮骆骆（镜003） |
| **和解/和解张力** | A B 同框+对视 | （本剧无明显和解） |

**示例**：

| 镜号 | 关系类型 | 镜头语言映射 |
|---|---|---|
| 镜 001 | 跟随（小仇→刘） | 小仇居中偏左紧随刘 |
| 镜 003 | 对抗（小仇→骆骆） | 小仇居中纵向攻击骆骆 |
| 镜 005 | 主导（刘→小仇骆骆） | 刘居中 + 其他人前景模糊 |
| 镜 008 | 反派对峙（小仇→刘） | 小仇凑上前 + 刘前景 |

**模板**：
```markdown
【角色心理关系】主导方：[谁主导本镜]。
              跟随方：[谁跟随/被动]。
              对抗关系：[如有对抗，谁对谁]。
              镜头语言：[景别/位置/景深如何映射心理关系]。
```

### 维度 10：戏剧张力（Dramatic Tension）

**定义**：这一镜的**张力等级**（1-10），观众感受到的紧张/期待/释放程度。

**为什么缺失**：现有 skill 把"张力"当作模糊感觉，没有量化到每镜。

**张力等级**：

| 等级 | 描述 | 镜头语言 | 示例 |
|---|---|---|---|
| 1-2 | 完全放松 | LS 远景 + 自然音 | 镜001 暖场（2） |
| 3-4 | 日常推进 | MS + 缓推 | 镜002 相撞前（4） |
| 5-6 | 紧张开始 | MCU + 节奏加速 | 镜002 相撞（5） |
| 7-8 | 高度紧张 | CU + 急推 + 音效 | 镜003 小仇咆哮（8） |
| 9-10 | 戏剧爆发/顶点 | CU→ECU + 多手段 | 镜009 普法高潮（9）/镜011 升华（10） |

**示例**：

| 镜号 | 张力等级 | 张力类型 |
|---|---|---|
| 镜 001 | 2 | 暖场 |
| 镜 002 | 5 | 紧张开始 |
| 镜 003 | 8 | 高度紧张 |
| 镜 005 | 7 | 教化推进 |
| 镜 009 | 9 | 戏剧高潮 |
| 镜 011 | 10 | 升华顶点 |

**模板**：
```markdown
【戏剧张力】等级：[1-10]。
              张力类型：[紧张/期待/释放/反转/升华]。
              张力曲线：[本镜张力如何变化，如"0-2秒3级→3-7秒9级"]。
              释放点：[如有释放，何时何方式]。
```

## 三、导演思维 10 维度整合示例（镜 005）

```markdown
镜 005 总裁长台词讲常识 9秒

【戏剧意图】让观众感受教化者的理性权威。
              触发条件：刘一鸣抬手打断，严肃普法。
              期待反应：观众认同教化者的价值观。

【情感弧线】刘一鸣：从"一脸无语"（延续镜004）→"理性权威教化者"（升弧）。
              小仇：从"咆哮嚣张"→"前景模糊被教化"（隐含的降弧）。
              骆骆：从"惊慌"→"前景模糊紧张"（持续紧张）。

【潜台词】刘一鸣说"什么叫'全球仅此一件的绝版高定'？我又不是明星走红毯"，
              潜台词是："别再用夸张叙事侮辱我的智商"。
              演员提示：眉头紧蹙+嘴角微抿带嘲讽。

【视线引导】主焦点：刘一鸣 MS 居中纵向（教化者本体）。
              次焦点：小仇/骆骆前景模糊（被教化者）。
              视线方向：观众看刘一鸣（权威源）。

【节奏设计】类型：舒缓（长台词教化）。
              节奏曲线：0-0.5秒紧凑（抬手打断）→0.5-9秒舒缓（教化展开）。
              镜头语言：MS 缓推 0.3 m/s + 自然音 + 眉头紧蹙。

【视觉隐喻】画面元素：刘一鸣抬手五指张开掌心向前。
              隐喻主题："教化的边界"（理性 vs 情绪）。
              主题服务：用物理动作传达"停止夸张"的教化。

【反差对比】与前镜：镜004 摘墨镜（爆发）→ 本镜 教化长台词（舒缓）（节奏反差）。
              与后镜：镜006 小仇呆若木鸡（持续舒缓）（节奏延续+情绪反差）。
              设计意图：爆发后用教化台词稳定观众情绪。

【主题表达】剧集主题："媒介素养"。
              本镜服务：用具体台词纠正"夸张叙事"（高定/打十辈子工）。
              主题强度：强（直接承载主题）。

【角色心理关系】主导方：刘一鸣（居中纵向）。
              跟随方：小仇 + 骆骆（前景模糊被教化）。
              对抗关系：刘（理性教化）vs 小仇（夸张叙事）。
              镜头语言：刘居中 + 其他人前景模糊。

【戏剧张力】等级：7。
              张力类型：教化推进（稳定紧张）。
              张力曲线：0-9秒稳定在 7 级（教化过程无突变）。
              释放点：本镜无释放，留到镜006（被教化反应）。
```

## 五、字数控制（与 v3.7 ≤1800 协调）

**新增导演意图字段会增加字数**：

| 维度 | 字数预算（每镜） | 优先级 |
|---|---|---|
| 戏剧意图 | 30-40 字 | 强制 |
| 情感弧线 | 30-40 字 | 强制 |
| 潜台词 | 30-40 字 | 强制（有对白时） |
| 视线引导 | 20-30 字 | 强制 |
| 节奏设计 | 20-30 字 | 强制 |
| 视觉隐喻 | 20-30 字 | 推荐 |
| 反差对比 | 20-30 字 | 强制 |
| 主题表达 | 20-30 字 | 强制 |
| 角色心理关系 | 20-30 字 | 推荐 |
| 戏剧张力 | 10-20 字 | 推荐 |

**总计**：强制 6 维度 × 25 字 ≈ 150 字 + 推荐 4 维度 × 25 字 ≈ 100 字 = 250 字

**对策**：从其他段让位（特别是"风格参考"段只保留 1-2 部，"调色管线"可缩短）
- v3.7 字数预算分配需要调整：导演意图段 200 字 + 其他段让位 50 字 = 总 1800 字仍守恒
