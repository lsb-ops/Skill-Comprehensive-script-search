# KB01 — 5段式提示词结构

> 即梦/可灵/Sora/Runway 通用提示词骨架
> 字数控制：单镜 ≤1800 字（v3.7 治本）/ ≤1700 字（合并镜）

---

## 一、设计哲学

5段式结构是**通用骨架**，适用于所有 AI 视频生成平台。它把"创作意图"翻译成"AI 可执行参数"，避免以下三类失败：

1. **关键词堆砌失败** — AI 抓不到主次
2. **描述模糊失败** — AI 生成画面漂移
3. **风格跳跃失败** — 多镜之间不像同一部剧

5段式不是"写法约束"，而是**信息密度分配法**：每段承担一个不可替代的信息层。

---

## 二、5段结构总览

| 段号 | 段名 | 字符占比 | 核心作用 | 不可省略信息 |
|---|---|---|---|---|
| 1 | 资产引用 | 10-15% | 锚定人物/道具/场景一致性 | 资产编号 + 延续关系 |
| 2 | 镜头参数 | 15-20% | 景别/焦距/运镜/画幅 | 12 核心参数（见 KB02）|
| 3 | 画面描述 | 35-40% | 时序化的视觉叙事 | 0-X秒分段动作 + 物理细节 |
| 4 | 光影氛围 | 10-15% | 色温/光比/色调 | 三点布光 + 色温曲线 |
| 5 | 技术参数 | 15-20% | 调色/参考/物理/合规 | LUT + 胶片颗粒 + 视觉参考 |

---

## 三、各段详解

### 段1：资产引用

**作用**：通过【资产编号】锚定人物/道具/场景，确保多镜一致性。

**必备元素**：
- 【人物_XXX：】（姓名 + 简述 + 延续镜号）
- 【道具_XXX：】（关键道具 + 位置 + 延续镜号）
- 【场景_XXX：】（场景图编号 + 简述）

**反例**（错误）：
```
资产引用
刘一鸣、小仇、骆骆
```
**正例**（正确）：
```
资产引用
【人物_刘一鸣：】【人物_小仇：画面外已被教化】【道具_墨镜：收于西装内袋延续镜004到010】【场景_弄堂内：】
```

**关键原则**：
- 不描述外貌（已用图片资产替代）
- 必须标注延续关系（"延续镜00X到00Y"）
- 画面外角色也要写（"画面外已被教化"）

---

### 段2：镜头参数

**作用**：定义画面物理规格，让 AI 知道"用哪只眼睛看"。

**12 核心参数**（见 KB02）：
- 时长 / 画幅 / 景别 / 焦距 / 运镜 / 机位 / 高度 / 视角 / 视点 / 构图 / 景深 / 帧率

**示例**：
```
镜头参数
时长7秒，画幅9比16竖屏，景别CU近景0到3.5秒，MS中景3.5到7秒。焦距85毫米CU段长焦浅景深，35毫米MS段。运镜固定机位CU段加缓Dolly Out MS段0.3米每秒后退。
```

**注意**：
- 多段运镜要分段标注（"X段+Y段+Z段"）
- 焦距变化要对应景别变化
- 运镜速度要给出具体数值（米/秒）

---

### 段3：画面描述

**作用**：核心叙事段，按时间顺序描述每个瞬间发生什么。

**必备元素**：
- 时序分段（"0到X秒"）
- 人物动作（含物理细节：速度/角度/肌肉）
- 表情变化（眉头/眼神/下颌）
- 道具交互
- 服装细节（自然褶皱/反光）

**示例**：
```
画面描述
0到3.5秒，CU近景85毫米长焦浅景深。资产人物小仇CU居中纵向，剧本原词瑟瑟发抖真实物理，肩膀加上臂加前臂的肌肉颤动真实物理v3.0强化每秒4到6次颤动频率。
3.5到4.5秒，剧本原词摇摇头叹了口气同框瞬间。资产人物小仇前景CU缩小为左侧三分之一，加资产人物刘一鸣MCU居中偏右。
```

**关键原则**：
- **时序优先**：0-X秒 → X-Y秒 → Y-Z秒
- **物理真实**：所有动作都要有物理量级（速度/角度/频率）
- **因果链**：动作→结果→下一个动作
- **剧本原词保留**：动作描写中保留原词加引号

---

### 段4：光影氛围

**作用**：定义情绪基调，让 AI 知道"这个场景是什么心情"。

**三要素**：
- **三点布光**：Key Light（主光）+ Fill Light（辅光）+ Back Light（轮廓光）
- **色温**：单位 K（开尔文）
- **光比**：主光:辅光（如 1:2 / 1:3）

**示例**：
```
光影氛围
主光Key Light，弄堂天光柔光4000K从画面左上方45度，强度90%延续镜009。辅光Fill Light，4500K冷蓝，强度30%延续镜009。轮廓光Back Light，5000K中性偏暖，强度80%延续镜009。色温4000K冷静科普美学延续镜009。光比1比2柔光。色调中性青橙加自然饱和。
```

**注意**：
- CUE 时刻必须显式标注色温跳变（如"色温跳变4000K到5500K"）
- 光比决定"强光"或"柔光"
- 色调决定整体情绪（青橙/暖金/冷蓝）

---

### 段5：技术参数

**作用**：后期管线 + 视觉参考 + 关键物理 + 合规约束。

**四要素**：
- **调色管线**：LUT 名称 + 胶片颗粒 + Halation
- **风格参考**：3-4 部参考电影/作品
- **关键物理**：本镜最核心的物理细节
- **关键合规**：字幕/文字禁忌

**示例**：
```
技术参数
调色管线Fuji Eterna加35毫米胶片颗粒无Halation去霸总美学延续镜009。风格参考功夫包租公被敲懵加海边的曼彻斯特抱紧自己加低俗小说教化者走开加布达佩斯大饭店配乐渐起。关键物理，v3.0强化瑟瑟发抖真实肌肉颤动每秒4到6次加剧本原词抱紧自己双臂交叉物理。关键合规AI画面严禁字幕文字未完待续七年等词所有字幕全部后期PR或AE叠加。
```

## 五、常见错误

### 错误1：资产引用写成"角色介绍"
❌ `刘一鸣：30岁男性，身高180cm，性格冷静。`
✅ `【人物_刘一鸣：】延续镜001到003。`

### 错误2：画面描述缺少时序
❌ `他走过来看着她笑了一下。`
✅ `0到2秒，他缓步走近。2到4秒，他在她面前停下，嘴角微动。`

### 错误3：光影氛围只写"柔和/昏暗"
❌ `光线柔和，氛围温馨。`
✅ `主光Key Light 4000K从左上方45度，强度80%。辅光Fill Light 4500K冷蓝，强度30%。色温4000K。光比1比2柔光。色调中性青橙。`

### 错误4：技术参数堆砌关键词
❌ `电影感、高级感、电影级画质、8K、HDR、杜比视界。`
✅ `调色管线Fuji Eterna加35毫米胶片颗粒无Halation。风格参考教父加天堂电影院。`

### 错误5：画面描述中描述人物外貌
❌ `刘一鸣身高180cm，黑色短发向后梳，戴着 Prada 墨镜。`
✅ `【人物_刘一鸣：】已在资产档案定义，画面中通过图片参考实现一致性。`

## 七、检验清单

写完一镜后，对照以下清单：

- [ ] 资产引用是否标注延续关系？
- [ ] 镜头参数是否覆盖 12 核心参数？
- [ ] 画面描述是否按时序分段？
- [ ] 光影氛围是否包含三点布光 + 色温 + 光比？
- [ ] 技术参数是否包含 LUT + 参考 + 合规？
- [ ] 总字数是否在 ≤1800 之间？（v3.7 治本）
- [ ] 有台词镜是否有【声音_角色：】引用？（v3.7 治本）
- [ ] 是否避免了人物外貌描述？
- [ ] 剧本原词是否 100% 保留？
