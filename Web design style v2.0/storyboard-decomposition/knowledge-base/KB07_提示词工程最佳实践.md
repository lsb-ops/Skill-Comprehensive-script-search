## 一、字数控制（≤1800 字 · v3.7 治本）

### 为什么 ≤1800 字？（v3.7 治本 · 基于 11 镜实测）

| 字数 | 效果 |
|---|---|
| < 1400 字 | 信息不足，AI 抓不到重点 |
| **1400-1800 字** | **v3.7 标准区间（信息密度最优）** |
| 1800-2200 字 | 偏多（v3.7 标准 ≤1800） |
| 2200-2400 字 | 过多（v3.7 强约束 ≤1800） |
| > 2400 字 | 失控（必须重写） |

**v3.7 治本变化**（从 2000-2400 → ≤1800）：
- ✅ AI 视频 API（即梦/可灵）注意力更集中
- ✅ v3.4 + v3.5 治本后信息密度已经够高
- ✅ 短视频单镜（5-8s）匹配度更高
- ✅ 实测：11 镜从 23255 字 → 14501 字（-37.6%）AI 一次成功率不降

### 字数分配（按 1800 目标）

| 段 | 字数 | 占比 |
|---|---|---|
| 资产引用（含声音） | 200-250 | 12% |
| 镜头参数 | 280-360 | 18% |
| 画面描述 | 650-780 | 40% |
| 光影氛围 | 220-280 | 14% |
| 技术参数 | 260-340 | 16% |

### 字数调节技巧

#### 太多（> 1800）
- 合并相似描述
- 删除重复修饰（"真实物理"/"真实物理声"等）
- 简化风格参考（保留 2-3 部）
- 删除"补充物理细节"段（已融入画面描述）
- 删除重复列表（"皮肤微血管红晕/自然毛孔/发根阴影/发丝单丝级非块状"）

#### 太少（< 1400）
- 增加物理细节
- 强化光影氛围
- 补充风格参考
- 细化时间分段

## 二、去 AI 感 7 大原则

### 原则1：避免关键词堆砌
❌ `电影感、高级感、电影级画质、8K、HDR、杜比视界、专业制作、奥斯卡级别`
✅ `Fuji Eterna 调色 + 35mm 胶片颗粒 + 风格参考教父`

**为什么**：AI 把关键词当作独立权重，堆砌导致画面混乱。

### 原则2：具体胜于抽象
❌ `光线柔和，氛围温馨。`
✅ `主光 Key Light 4000K 从左上方 45 度，强度 80%。辅光 Fill Light 4500K 冷蓝，强度 30%。色温 4000K。光比 1:2 柔光。`

**为什么**：具体描述让 AI 有锚点，抽象描述让 AI 自由发挥。

### 原则3：物理量级优先
❌ `他走过来。`
✅ `他缓步走近，1.5 米/每秒真实行人速度，皮鞋踏步声真实石板路触感。`

**为什么**：物理量级让 AI 模拟真实物理。

### 原则4：风格参考具体
❌ `像电影一样`
✅ `风格参考教父 + 天堂电影院 + 低俗小说`

**为什么**：具体电影让 AI 有视觉锚点。

### 原则5：避免"AI 套话"
❌ `画面精美、构图严谨、色彩饱和、电影感强烈`
✅ 删除这些词，让结构和具体参数替代。

**为什么**：AI 套话浪费字数，无信息量。

### 原则6：时序分段必须
❌ `他看着她笑了一下，然后摇摇头走了。`
✅ `0到3秒，他缓步走近。3到5秒，他在她面前停下，嘴角微动。5到7秒，他摇摇头叹了口气，转身走开。`

**为什么**：时序分段让 AI 按时间生成，避免混乱。

### 原则7：照片级而非"美图"
❌ `美轮美奂的风景`
✅ `远景 ELS，焦距 24mm，现代摩天楼玻璃幕墙反光 + 红砖雕花立面 + 自然逆光`

**为什么**："美"是抽象词，具体场景描述让 AI 模拟真实。

## 三、清晰度提升技巧

### 技巧1：使用【资产编号】
```
【人物_刘一鸣：】
【道具_墨镜：】
【场景_弄堂尽头：】
```
**效果**：锚定一致性，AI 不会跑偏。

### 技巧2：使用"延续镜00X到00Y"
```
【道具_墨镜：】（收于西装内袋延续镜004到010）
```
**效果**：跨镜一致性。

### 技巧3：使用具体电影名
```
风格参考：教父 + 天堂电影院 + 低俗小说
```
**效果**：AI 有视觉参考库。

### 技巧4：使用专业术语
```
Key Light, Fill Light, Back Light
MS, CU, ECU
Dolly In, Dolly Out, Pan
Fuji Eterna, Halation
```
**效果**：AI 在大量训练数据中识别这些术语。

### 技巧5：使用数字编号
```
0到3秒，5到8秒，2.5到5.3秒
```
**效果**：精确时序。

### 技巧6：使用对比结构
```
色温 4000K → 5500K（跳变）
光比 1:2 → 1:∞（柔光 → 剪影）
```
**效果**：变化方向明确。

### 技巧7：使用"严禁"和"必须"
```
AI 画面严禁字幕文字
所有字幕后期 PR 或 AE 叠加
```
**效果**：负面约束，防止 AI 生成错误内容。

## 四、常见陷阱

### 陷阱1：描述人物外貌
❌ `刘一鸣身高 180cm，黑色短发向后梳，戴着 Prada 墨镜。`
✅ `【人物_刘一鸣：】（图片资产已在资产管理中定义）`

**原因**：人物一致性通过图片资产而非文字描述实现。

### 陷阱2：使用方括号/尖括号错误
❌ `[资产人物-XXX]`  ← 不规范
✅ `【人物_XXX：】` ← 中文规范

### 陷阱3：时序混乱
❌ `0到5秒，他走开。3到7秒，他叹气。` ← 时间重叠
✅ `0到3秒，他叹气。3到7秒，他走开。` ← 时间连续

### 陷阱4：物理量级缺失
❌ `他快速走开。`
✅ `他大步走开，1.5 米/每秒真实行人速度。`

### 陷阱5：风格参考过多
❌ 风格参考：教父 + 天堂电影院 + 低俗小说 + 布达佩斯大饭店 + 阿甘正传 + 辛德勒名单 + ...
✅ 风格参考：教父 + 天堂电影院 + 低俗小说（3 个）

### 陷阱6：色温光比无数字
❌ `光线比较冷`
✅ `色温 4000K，光比 1:3`

### 陷阱7：画面描述无时序
❌ `他走过来，她哭了，他递纸巾给她，她接过去。`
✅ `0到3秒，他走过来。3到5秒，她哭了。5到6秒，他递纸巾给她。6到7秒，她接过去。`

### 陷阱8：字幕违规
❌ 画面中包含"未完待续"等文字
✅ `AI 画面严禁字幕文字，所有字幕后期 PR 或 AE 叠加。`

## 五、剧本原词保留规则

### 必须保留
- **对白 100% 不改**：`她说："你怎么才来？"` 一字不改。
- **舞台指示**：动作、表情、语气全部保留。
- **关键术语**：剧本中定义的关键概念。

### 标注方式
```
剧本原词 + <原词内容>
```

### 示例
```
剧本原词瑟瑟发抖真实物理，肩膀加上臂加前臂的肌肉颤动。
剧本原词摇摇头叹了口气。
剧本原词转身大步走开。
```

### 反例（错误）
❌ `他吓得发抖`
✅ `剧本原词瑟瑟发抖真实物理`

**原因**：AI 不识别"吓得发抖"这种文学描述，识别"瑟瑟发抖+真实物理"具体动作。

## 十、v3 强化规范（2026-06-17 升级）

> 基于对 11 个分镜 txt 的逐字审阅修复。

### 10.1 `v3.0强化` 信号锚点规则

| 错误 | 正确 |
|---|---|
| 26-37 次/镜（v2.0 现状） | ≤ 2 次/镜（标题段 + CUE 段各 1 次） |

**原因**：T5-XXL / CLIP-L 文本编码器 token 数 77-256 上限，高频词被 AI 当**无意义 token**忽略，失去信号作用。

### 10.2 `真实物理` 抽象词替换规则

| 错误 | 正确 |
|---|---|
| `真实物理`（30+ 次/镜） | `G=-9.8 m/s²` / `角动量滞后 0.3s` / `Hertz 接触 0.02s` |
| `真实镜头物理` | `Dolly In 0.3米每秒` / `缓推 0.3米每秒` |
| `真实物理行走` | `1.5米每秒真实行人速度` |
| `真实物理反作用力` | `反作用力 0.2米推后` |

**原因**：AI 不识别"真实物理"作为信号，但能识别**具体数字+单位**作为权重 token。

**保留词**：`真实物理引擎渲染`（即梦 hot-token）每镜 1-2 次保留。

### 10.3 T-stop 必加规则

| 焦段 | T-stop | 适用 |
|---|---|---|
| 24mm | T8 | LS 远景深景全幅 |
| 35mm | T2.8 | MS 标准中景浅景深 |
| 50mm | T2 | MCU 中近景浅景深 |
| 85mm | T1.4 | CU 近景极浅景深 |
| 135mm | T1.4 | ECU 极特写强压缩+浅景深 |

**缺失后果**：AI 默认深景深，背景清晰 = 视频感重。

### 10.4 抽象词黑名单

| 禁用 | 原因 |
|---|---|
| 蒙太奇美学 / 教化者美学 / 冷静科普美学 | 抽象标签，AI 自由发挥导致混乱 |
| 极具压迫感 / 戏剧光晕 / 蒙太奇霸气美学 | 抽象感觉描述 |
| 美轮美奂 / 震撼 / 极致 / 唯美 | 无信号量 |
| AI 后期逆光晕 | "AI 后期"是反信号 |
| AI 后期 / AI 渲染 / AI 增强 | 元注释无意义 |
| 8K / HDR / 杜比视界 / 电影级 | KB07 已禁，堆砌稀释主语 |

**例外保留**：
- `升华美学`（CUE③ 升华镜可用 1 次）
- `真实物理引擎渲染`（即梦 hot-token，每镜 1-2 次）

### 10.5 色温与时间匹配

| 场景 | 色温 |
|---|---|
| 日景晴天 | 5000-5500K |
| 日景阴天 | 6500-7000K |
| 黄昏 / Golden Hour | 3000-3500K |
| 夜景暖内（钨丝） | 3200K |
| 夜景冷街（街灯） | 4500-5000K |
| 蓝调时刻 | 8000-10000K |
| 冷静科普（CUE②） | 4000K |
| 升华（CUE③） | 5500K |

**错误**：日景外景写 4500K → AI 误读为黄昏。

### 10.6 风格参考 ≤ 3 部

| 错误 | 正确 |
|---|---|
| 教父+花样年华+低俗小说+大话西游+功夫（5 部） | 教父+花样年华（2 部） |
| `教父+花样年华+低俗小说` | `风格参考：教父；副参考：花样年华` |

### 10.7 Halation 日常戏关闭

| 场景 | Halation | 原因 |
|---|---|---|
| 日常对话戏 | 关 | 日常开 = 塑料感 |
| 夜场外景 | 开（弱） | 模拟胶片乳剂层反射 |
| 逆光/背光 | 开（中） | 真实物理反射 |
| CUE③ 升华 | 开（强） | 升华美学 |

### 10.8 v3 升级前后对比

| 指标 | v2.0 | v3.0 | 变化 |
|---|---|---|---|
| v3.0强化 | 323 次 | 1 次 | ↓99.7% |
| 真实物理（抽象词） | 347 次 | 46 次 | ↓86.7% |
| T-stop | 0 次 | 45 次 | ↑from 0 |
| 抽象美学词 | ~30 | 1 次 | ↓96.7% |
| 字数合规率 | 82% | 100% | ↑18% |
