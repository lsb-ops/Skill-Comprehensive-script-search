# KB05 — CUE 设计原则

> 戏剧断点（Climactic Unit of Emotion）
> 短视频注意力管理核心

---

## 一、什么是 CUE

**CUE（Climactic Unit of Emotion，戏剧情绪单元）**是短视频中通过**视听手段**（非剧情）制造的戏剧断点。

### 与"剧情转折"的区别

| 维度 | CUE（戏剧断点） | Plot Twist（剧情转折） |
|---|---|---|
| 本质 | 视听语言 | 故事事件 |
| 表现 | 音乐/光影/运镜/色温 | 角色命运/关系变化 |
| 感受 | 情绪冲击 | 认知冲击 |
| 举例 | 教父音乐响起 | 主角身份暴露 |

### CUE 的作用
- 在短视频极短时间内制造**情绪记忆点**
- 让观众"看到某个瞬间被击中"
- 弥补短视频叙事时间不足
- 提升作品的"电影感"

---

## 二、CUE 三元素

### 元素1：色温跳变
- 单位：开尔文 K
- 操作：在 0.5-1 秒内完成色温跳变
- 视觉：冷暖切换、情绪切换

### 元素2：光比变化
- 操作：光比从 1:2 变为 1:4 或 1:∞
- 视觉：柔光 → 强光 → 剪影

### 元素3：音效/配乐触发
- 操作：关键音效 + 配乐渐起/止
- 视觉：听觉同步强化

### 元素4（可选）：运镜变化
- 操作：缓推 → 急推，或固定 → 缓拉
- 视觉：物理逼近或抽离

### 元素5（可选）：构图变化
- 操作：三分 → 中心，对称 → 对角线
- 视觉：构图强化情绪

---

## 三、典型 CUE 类型

### CUE①：开篇风格切换（前 0-10 秒）

**目的**：建立作品风格基调，从"日常"切到"作品世界"。

**操作**：
- 色温跳变：2700K → 4500K（暖场 → 自然）
- 配乐：从无到渐起
- 光比：1:1 → 1:2（平光 → 柔光）

**适用**：开篇镜、风格定调

**示例（霸总剧）**：
```
0到3秒：暖场，色温 2700K，光比 1:1，无配乐。
3到5秒：CUE①，色温跳变 2700K → 4500K，配乐渐起，光比变为 1:2。
5到8秒：自然基调建立，MS 中景，主光从左上方 45 度。
```

### CUE②：中段教化升级 / 情绪升级（30-40 秒处）

**目的**：从"叙事"升级到"教化/震撼"，情绪高点。

**操作**：
- 色温跳变：5000K → 4000K（自然 → 冷静）
- 配乐：从轻到重 / 静默 0.3 秒后爆炸
- 光比：1:2 → 1:3（柔光 → 强光）

**适用**：中段高潮、情绪爆点、揭示时刻

**示例（霸总剧普法）**：
```
0到3秒：日常对话，色温 5000K，光比 1:2。
3到3.3秒：CUE②，刹车声 + 静默 0.3 秒（关键音效）。
3.3到7秒：色温跳变 5000K → 4000K（冷静科普），配乐起，光比变为 1:3，主光变冷。
```

### CUE③：升华（70-80 秒处）

**目的**：从"教化/叙事"升华到"主题/普世价值"，情绪最高点。

**操作**：
- 色温跳变：4000K → 5500K（冷静 → 暖金）
- 配乐：高峰 + 内心 OS
- 光比：1:3 → 1:∞（柔光 → 纯逆光剪影）
- Halation：开
- AI 后期逆光晕强化

**适用**：结尾、主题升华、背影、慢动作

**示例（霸总剧普法升华）**：
```
0到2秒：CUE③，色温跳变瞬间 4000K → 5500K（核心真实物理过渡不是渐变）。
2到5秒：缓推 + 配乐高峰 + 内心 OS。
5到8秒：缓拉 + 背影渐远 + Halation 开 + AI 后期逆光晕。
```

## 四、CUE 设计原则

### 原则1：不超过 3 个
短视频注意力有限，3 个 CUE 是上限。

| 时长 | CUE 数量 | 位置 |
|---|---|---|
| 30-60 秒 | 1-2 个 | 开头 + 结尾 |
| 60-90 秒 | 2-3 个 | 开头 + 中段 + 结尾 |
| 90-180 秒 | 3-4 个 | 间隔 30-60 秒 |

### 原则2：必须有差异化
每个 CUE 必须有不同的视听特征，避免重复。

| CUE | 色温方向 | 光比方向 | 配乐方向 |
|---|---|---|---|
| CUE① 开篇 | 暖→自然 | 1:1 → 1:2 | 无→渐起 |
| CUE② 中段 | 自然→冷 | 1:2 → 1:3 | 轻→重 |
| CUE③ 升华 | 冷→暖 | 1:3 → 1:∞ | 重→高峰 |

### 原则3：必须有物理过渡
CUE 不是"突变"，而是"真实物理过渡"。

| 错误 | 正确 |
|---|---|
| 突然黑屏 | 0.3 秒渐黑 |
| 突然切换色温 | 0.5-1 秒色温物理跳变（真实灯光变化） |
| 突然静音 | 0.3 秒静默后渐起 |

### 原则4：必须有视觉锚点
CUE 时刻要有一个标志性视觉，让观众记住。

| CUE | 视觉锚点 |
|---|---|
| CUE① | 标志性的开场构图 / 配乐前奏 |
| CUE② | 关键音效（刹车声/敲门声） + 静默 |
| CUE③ | 标志性逆光剪影 + Halation |

### 原则5：必须服务于主题
每个 CUE 必须推动主题表达，不能为 CUE 而 CUE。

| CUE | 主题服务 |
|---|---|
| CUE① | "欢迎进入作品世界" |
| CUE② | "这里是关键，请注意" |
| CUE③ | "这是作品想说的话" |

## 五、CUE 的视觉化表达

### 在提示词中标注 CUE

每镜应明确标注是否包含 CUE，以及是哪种 CUE。

```
镜011 背影光环升华片尾升华CUE 8秒
↑                ↑
标题含 CUE      CUE 类型
```

### CUE 在画面描述中的体现

```
画面描述
0到2秒：CUE③色温跳变瞬间。色温 4000K → 5500K 升华CUE核心跳变从画面正前方逆光强度100%到120%。
2到5秒：缓推 + 配乐高峰 + 内心 OS 起。
5到8秒：缓拉 + 背影渐远 + Halation 开。
```

### CUE 在光影氛围中的体现

```
光影氛围
0到2秒：主光 Key Light 4000K → 5500K 升华CUE核心跳变。
2到8秒：主光 Key Light 5500K 升华暖金 v3.0 强化逆光强度 120%。
光比 1:∞（剪影）。
色调：升华暖金 普法先锋的光环美学。
```

### CUE 在技术参数中的体现

```
技术参数
关键物理：升华CUE色温跳变 + 逆光剪影 + 背影渐远 1.5 米/每秒 + 缓Dolly Out + AI后期逆光晕。
风格参考：教父背影收束 + 阿拉伯的劳伦斯逆光剪影 + 天堂电影院结尾升华 + 功夫如来神掌光环。
```

## 六、各风格的 CUE 模式

### 古早霸总
- CUE①：配乐前奏 + 慢动作
- CUE②：戏剧性独白 + 配乐爆发
- CUE③：逆光剪影 + 暖金光环

### 都市情感
- CUE①：城市天际线 + 钢琴前奏
- CUE②：雨中独白 + 弦乐
- CUE③：拥抱特写 + 暖光渐亮

### 古装江湖
- CUE①：山水空镜 + 鼓声
- CUE②：剑光 + 配乐爆发
- CUE③：夕阳剪影 + 笛声

### 科幻未来
- CUE①：电子音效 + 数字故障
- CUE②：能量波动 + 配乐爆发
- CUE③：光速推进 + 静音

### 校园青春
- CUE①：阳光教室 + 吉他前奏
- CUE②：奔跑 + 配乐爆发
- CUE③：黄昏剪影 + 慢动作

### 悬疑推理
- CUE①：城市夜景 + 低频
- CUE②：发现证据 + 配乐爆发
- CUE③：真相揭示 + 静音

### 战争军事
- CUE①：战场空镜 + 鼓声
- CUE②：爆炸 + 配乐爆发
- CUE③：胜利剪影 + 国歌

### 家庭伦理
- CUE①：家庭日常 + 钢琴
- CUE②：争吵 + 配乐爆发
- CUE③：拥抱 + 暖光渐亮

## 七、CUE 与配乐协同

### 配乐结构

| 阶段 | 配乐状态 | 时长占比 |
|---|---|---|
| 开场 | 无 / 极轻 | 0-10% |
| 建立 | 渐起 | 10-30% |
| 叙事 | 中等 | 30-70% |
| CUE 爆发 | 高峰 | 70-80% |
| 收束 | 渐弱 | 80-100% |

### 配乐音量（dB）

| 阶段 | 音量 |
|---|---|
| 静默 | -∞ dB |
| 环境音 | -40 dB |
| 配乐渐起 | -30 dB |
| 配乐中等 | -20 dB |
| 配乐高潮 | -10 dB |
| 配乐高峰 | -6 dB |

## 八、CUE 检验清单

每镜完成后检查：

- [ ] 是否标注了 CUE 类型？
- [ ] CUE 是否按时序分段？
- [ ] 色温跳变是否有具体 K 值？
- [ ] 光比变化是否有具体方向？
- [ ] 配乐变化是否有具体描述？
- [ ] 是否有视觉锚点？
- [ ] CUE 是否服务于主题？
- [ ] 全剧 CUE 总数是否超过 3 个？

---

## 九、避免的 CUE 错误

### 错误1：CUE 过多
3 个以上 CUE → 观众"麻木"，失去冲击。

### 错误2：CUE 无差异
3 个 CUE 都用"配乐爆发" → 重复。

### 错误3：CUE 不服务主题
为炫技而 CUE → 作品空洞。

### 错误4：CUE 时机不准
在不需要情绪爆发的点强行 CUE → 突兀。

### 错误5：CUE 物理过渡不真实
突然黑屏 → 廉价感。
0.3 秒渐黑 → 电影感。

## 十、CUE 在 AI 提示词中的标准模板

```
镜XXX <主题>CUE <CUE类型> <时长>秒

[光影氛围 段]
0到X秒：<CUE前状态>色温 YYYYK，光比 1:Z
X到X+0.5秒：CUE触发，色温跳变 YYYYK → WWWK，光比 1:Z → 1:W
X+0.5到Y秒：<CUE后状态>色温 WWWK，光比 1:W，配乐高峰

[技术参数 段]
关键物理：CUE色温跳变 + <CUE视觉效果>
风格参考：<电影1> + <电影2> + <电影3>
```
