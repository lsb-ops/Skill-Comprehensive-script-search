# M1 — 剧本解析模块

> 阶段1：将任意格式剧本解析为结构化数据
> 输入：任意剧本（小说/大纲/分场/完整剧本）
> 输出：场景列表+人物清单+5幕结构+剧本原词+动作清单

---

## 一、模块目标

把任意格式的剧本（小说/故事大纲/分场剧本/完整剧本）解析为后续阶段需要的结构化数据：

1. **场景列表**：每个场景的时间、地点、人物
2. **人物清单**：人物角色、关系、定位
3. **5幕结构**：建制/触发/反转一/反转二/收束
4. **剧本原词**：100% 保留的对白+舞台指示
5. **戏剧动作清单**：每个动作的物理细节

---

## 二、输入

任意格式剧本，支持以下类型：

| 类型 | 示例 |
|---|---|
| 小说 | 一段叙述性文字 |
| 故事大纲 | 简短的剧情摘要 |
| 分场剧本 | 按场次组织的剧本 |
| 完整剧本 | 包含对白+舞台指示的完整剧本 |
| 漫画脚本 | 画面+文字描述 |

## 四、工作步骤

### 步骤 1：通读剧本

**目标**：理解整体故事走向

**操作**：
- 完整阅读剧本 1-2 遍
- 标注关键事件
- 标注情绪转折点
- 标注 CUE 候选位置

### 步骤 2：场景切分

**目标**：把剧本切分为独立场景

**切分依据**：
- 时间变化（早→中→晚）
- 地点变化（家→公司→咖啡厅）
- 人物变化（核心人物更换）

**输出**：场景列表（输出1）

### 步骤 3：人物识别

**目标**：列出所有角色

**识别要素**：
- 姓名
- 性别
- 年龄
- 角色定位（主角/配角/反派）
- 关键外貌特征（用于图片资产）
- 性格特征
- 弧光（人物在剧中的成长/变化）

**输出**：人物清单（输出2）

### 步骤 4：5幕结构识别

**目标**：把剧本对应到经典 5 幕结构

**5 幕结构**：
- **建制（Setup）**：建立日常、人物登场
- **触发（Inciting Incident）**：打破日常的事件
- **反转一（First Plot Point）**：主角的第一次冲击
- **反转二（Second Plot Point / Climax）**：高潮，主角做出选择
- **收束（Resolution）**：主题升华

**输出**：5幕结构（输出3）

### 步骤 5：剧本原词提取

**目标**：把所有对白+舞台指示 100% 提取

**关键原则**：
- 一字不改
- 标注来源（哪一幕/哪一场）
- 标注类型（对白/舞台指示）
- 标注说话人（对白）

**输出**：剧本原词（输出4）

### 步骤 6：戏剧动作清单

**目标**：把剧本动作转换为时序化的物理动作

**转换规则**：
- 把文学描述转换为物理动作
- 把抽象动作具体化（"走过来" → "缓步走近，1.5 米/每秒真实行人速度"）
- 把单一动作分解为多步（如"他接过名片看了一眼" → "他伸手接过 + 低头看名片 + 表情变化"）

**输出**：戏剧动作清单（输出5）

## 五、关键原则

### 原则1：剧本原词 100% 保留

**对白**一字不改。
**舞台指示**完整保留。

这是后续所有阶段的"事实基础"。一旦修改，作品就失去了"忠于剧本"的根基。

### 原则2：物理细节前置

在解析阶段，就把抽象动作具体化为物理细节：

```
原剧本：他走过来。

解析后：
- 动作：缓步走近
- 速度：1.5 米/每秒真实行人速度
- 步态：自然
- 音效：皮鞋踏步声真实石板路触感
```

### 原则3：场景不要超过 4 个

短视频时间有限，场景过多会拖慢节奏。

| 剧本规模 | 推荐场景数 |
|---|---|
| 60 秒 | 1-2 个 |
| 80 秒 | 2-3 个 |
| 90-180 秒 | 3-4 个 |

### 原则4：人物不要超过 5 个

短视频人物过多会分散注意力。

| 剧本规模 | 推荐人物数 |
|---|---|
| 60 秒 | 2-3 个 |
| 80 秒 | 3-4 个 |
| 90-180 秒 | 4-5 个 |

### 原则5：5 幕结构必须清晰

即使剧本很短，也要识别出"建制/触发/反转一/反转二/收束" 5 个阶段。

每个阶段至少 1 镜，最长不超过 30 秒。

## 七、常见错误

### 错误1：场景切分过细

❌ 把每个对话都作为一个场景
✅ 按"时间+地点+人物组"切分场景

### 错误2：人物清单过于详细

❌ 列出人物的所有背景故事
✅ 只列出与视觉相关的信息（外貌/服装/关键特征）

### 错误3：5幕结构强行套用

❌ 短剧本只有 3 幕结构，硬套 5 幕
✅ 根据剧本实际结构划分，可以是 3-5 幕

### 错误4：剧本原词改动

❌ "让开。" → "让开吧。"（添加"吧"）
✅ "让开。"（一字不改）

### 错误5：动作描述文学化

❌ "他看起来很害怕"
✅ "剧本原词瑟瑟发抖真实物理，肩膀加上臂加前臂的肌肉颤动真实物理"

- [ ] 人物数量是否合理（≤5）？
- [ ] 5幕结构是否清晰？
- [ ] 剧本原词是否 100% 保留？
- [ ] 戏剧动作是否物理化？
- [ ] 是否标注了 CUE 候选位置？
