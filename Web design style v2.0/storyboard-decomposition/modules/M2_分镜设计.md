# M2 — 分镜设计模块

> 阶段2：将解析后的剧本转换为分镜大纲
> 输入：M1 输出 + 风格预设
> 输出：分镜大纲 + 场景主控档案 + 衔接表 + CUE 清单 + 色温曲线 + **场景拓扑图（M10 强制新增）**

> **版本**：v1.1（2026-06-17）—— M10 新增步骤 9 场景拓扑图
> - 强制产出：`00_总览/00_场景拓扑图.md`
> - 内容：场景平面图 + 默认站位 + 机位方向约定
> - 用途：每镜 prompt 顶部"场景空间坐标"段引用

---

## 一、模块目标

把 M1 输出的结构化数据，转换为分镜设计：

1. **分镜大纲**：每镜的时长、景别、运镜、动作概要
2. **场景主控档案**：每个场景的统一定义
3. **衔接表**：跨镜位置关系
4. **CUE 清单**：3 个 CUE 的具体位置和效果
5. **色温曲线**：全剧色温变化

---

## 二、输入

- M1 输出的 5 个数据（场景列表/人物清单/5幕结构/剧本原词/动作清单）
- 风格预设（styles/STYLE_xxx.md）

---

## 四、工作步骤

### 步骤 1：加载风格预设

从 `styles/STYLE_xxx.md` 读取：
- 视觉基线（色温曲线、光比曲线、色调）
- CUE 设计（数量、模式）
- 视觉参考（电影列表）
- 标配元素（道具、表情）
- BGM 风格
- 典型结构（幕数、镜数、总时长）

### 步骤 2：镜数规划

根据剧本规模和风格预设，规划总镜数。

| 时长 | 推荐镜数 | 平均镜长 |
|---|---|---|
| 30 秒 | 4-6 | 5-8 秒 |
| 60 秒 | 8-10 | 6-8 秒 |
| 80 秒 | 10-12 | 7-8 秒 |
| 90-180 秒 | 12-20 | 6-10 秒 |

**霸总剧典型**：80 秒 / 11 镜 / 平均 7.3 秒

### 步骤 3：时长分配

按 5 幕结构分配时长：

| 幕 | 时长占比 | 80 秒剧本 |
|---|---|---|
| 建制 | 15-20% | 12-16 秒 |
| 触发 | 15-20% | 12-16 秒 |
| 反转一 | 15-20% | 12-16 秒 |
| 反转二 | 25-35% | 20-28 秒 |
| 收束 | 10-15% | 8-12 秒 |

### 步骤 4：CUE 设计

按风格预设的 CUE 模式，设计 3 个 CUE 的具体位置。

| CUE | 位置 | 效果 |
|---|---|---|
| CUE① | 开篇 0-10 秒 | 建立风格基调 |
| CUE② | 中段 30-40 秒 | 情绪升级 |
| CUE③ | 结尾 70-80 秒 | 主题升华 |

**注意**：
- CUE 之间间隔至少 25 秒
- CUE 必须有差异化
- CUE 必须服务主题

### 步骤 5：色温曲线设计

设计全剧色温变化曲线：

**基础曲线**：
- 建制：4500K-5000K（自然）
- 触发：5000K（自然）
- 反转一：4500K（中性偏冷）
- 反转二：4000K（冷静科普）
- 收束：5500K（升华暖金）

**CUE 时刻**：
- CUE①：2700K → 4500K（暖场 → 自然）
- CUE②：5000K → 4000K（自然 → 冷静）
- CUE③：4000K → 5500K（冷静 → 升华）

### 步骤 6：场景主控档案

为每个场景生成主控档案：

**必备元素**：
- 基本信息（编号、名称、时间、地点、覆盖镜号）
- 视觉定义（场景图、背景元素、主导色温、主导光比、主导色调）
- 镜头规格（画幅、机位、高度）
- 道具清单
- 人物清单

### 步骤 7：分镜大纲

为每镜设计：
- 镜号
- 时长
- 画幅
- 景别（MS / CU / ECU / 等）
- 焦距
- 运镜
- 5幕位置
- CUE 关联
- 内容概要
- 关键道具
- 关键对白
- 物理细节

### 步骤 8：衔接表

为每镜衔接设计：
- 上镜结束位置（人物位置、景别、动作）
- 本镜开始位置（人物位置、景别、动作）
- 衔接方式（同位置续拍 / 跳轴 / 拉远建立 / 推近聚焦）

## 五、关键原则

### 原则1：场景主控档案是"单一事实来源"

所有镜共享同一份场景定义。一旦修改，所有相关镜都要同步更新。

### 原则2：每镜时长 5-12 秒

| 镜长 | 适用 |
|---|---|
| 5-7 秒 | 标准镜 |
| 7-9 秒 | 复杂动作/对白 |
| 9-12 秒 | 长镜头 |
| > 12 秒 | 拆分 |

### 原则3：衔接优先考虑"同场景续拍"

同场景内的镜头，**优先考虑连续生成**（一次性生成多镜），保证人物一致性。

**短剧 AI 生成关键**：同场景多镜合并为一次生成。

### 原则4：跨场景必须有"过渡"

跨场景时，必须用以下方式之一过渡：
- 空镜过渡（LS 远景建立新场景）
- 道具过渡（用道具关联两场景）
- 色温突变（暗示时间/地点变化）

### 原则5：CUE 不要超过 3 个

短视频注意力有限，3 个 CUE 是上限。

### 原则6：色温曲线必须服务主题

冷→暖 = 教化/胜利
暖→冷 = 危机/恶化
平→极端 = 升华
