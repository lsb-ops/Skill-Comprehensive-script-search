# M1.5 — 戏剧意图解读模块（v3.8 治本 · 导演思维强化）

> **阶段 1.5**：在 M1（剧本解析）和 M2（分镜设计）之间
> **目的**：把剧本的"心理/情感/主题"翻译成"导演意图"，作为 M2 分镜设计的输入
> **输入**：M1 输出的剧本结构化数据
> **输出**：戏剧意图总纲 + 5 幕情感弧线 + 潜台词解读

> **版本**：v1.0（2026-06-17）
> **关联**：KB13 导演思维体系 + M_角色调用机制.导演解读阶段 + M2 步骤 7 强化

---

## 一、模块目标

### 为什么要 M1.5

```
剧本：心理/情感/主题（抽象层）
    ↓ [M1.5 翻译：导演意图] ← 缺失！
分镜：画面/声音/节奏（具象层）
```

**没有 M1.5 的问题**：
- LLM 在 M2 直接做技术分解，但不知道"为什么这样拍"
- 生成的画面"形式正确但叙事空洞"
- AI 视频生成结果"画面精美但缺乏导演意图"

**M1.5 的价值**：
- 把导演的"心理翻译"显化为结构化文档
- M2 分镜设计有明确的"导演意图输入"
- AI 视频生成时，每镜都有"导演意图锚点"

---

## 二、模块输入

```
M1 输出：
- 场景列表
- 人物清单
- 5 幕结构（建制/触发/反转一/反转二/收束）
- 剧本原词
- 动作清单
- 风格预设
```

## 三、模块输出

### 3.0 输出路径

```
{output_root}/{剧本名}/00_总览/00_戏剧意图总纲.md
```

### 3.1 输出结构（6 大模块）

```markdown
## 戏剧意图总纲

### 1. 剧集核心主题
[一句话核心主题]

### 2. 5 幕情感弧线
[每幕的情绪曲线 + 关键词]

### 3. 5 幕戏剧意图

### 4. 角色情感轨迹
[每个角色在全剧的心理变化轨迹]

### 5. 潜台词解读
[关键对白的潜台词分析]

### 6. 视觉隐喻清单
[剧集核心视觉隐喻]

### 7. 节奏曲线设计
[全剧的紧张/舒缓节奏曲线]

### 8. 主题表达强度分布
[每幕的主题承载强度]
```

## 四、工作步骤

### 步骤 1：调用导演角色 · 解读剧本

```markdown
## 🎬 导演解读阶段（v3.8 新增）

你是[剧名]的导演。请基于以下剧本给出导演意图解读：

【剧本概要】{M1 输出的 5 幕结构 + 关键对白 + 关键动作}
【风格预设】{古早霸总/都市情感/...}

请输出：
1. 剧集核心主题（1 句话）
2. 5 幕情感弧线（每幕情绪关键词）
3. 5 幕戏剧意图（每幕让观众感受什么）
4. 主角心理轨迹（情感起点 → 终点）
5. 3 个核心潜台词（最有张力的对白）
6. 3 个核心视觉隐喻（最有冲击力的画面）
7. 全剧节奏曲线（紧张/舒缓分布）
8. 主题强度分布（每幕对主题的承载）
```

**输出格式**：
```markdown
### 1. 剧集核心主题
当权势遇到常识，理性如何教化浮夸。

### 2. 5 幕情感弧线
- 建制：冷漠/压抑（霸总滤镜下的人物登场）
- 触发：意外/紧张（日常打破霸总）
- 反转一：教化/理性（刘一鸣开始教化）
- 反转二：严肃/权威（刘一鸣深入普法）
- 收束：升华/回味（教化完成 + 孤独的背影）

### 3. 5 幕戏剧意图
- 建制：让观众感受霸总的压迫感，建立角色权威
- 触发：让观众感受日常打破霸总的紧张感
- 反转一：让观众感受教化者的理性权威
- 反转二：让观众感受普法者的严肃权威
- 收束：让观众感受升华的孤高（教化者也是孤独者）

### 4. 主角心理轨迹
- 刘一鸣：冷漠霸总 → 教化者 → 普法者 → 升华者
- 小仇：嚣张 → 被教化 → 瑟瑟发抖
- 骆骆：惊慌 → 持续紧张 → 逃离

### 5. 3 个核心潜台词
- 镜005 "什么叫'全球仅此一件的绝版高定'？" → "别再用夸张叙事侮辱我的智商"
- 镜009 "公民的个人信息受法律保护" → "你刚才想做的事是违法的"
- 镜011 "这届助理真难带一点媒介素养都没有" → "教化是孤独的，但值得"

### 6. 3 个核心视觉隐喻
- 皮鞋踏步 + 张园石库门 → 权势踏入旧上海
- 摘墨镜 + 真实表情 → 撕下假面，回归真实
- 背影剪影 + 摩天楼 → 个体在时代巨浪中的孤独与坚守

### 7. 全剧节奏曲线
建制（紧张2）→ 触发（紧张5）→ 反转一（教化7）→ 反转二（普法9）→ 收束（升华10）
[低开高走，渐进式教化，最后升华]

### 8. 主题强度分布
- 建制：弱（仅建立反差）
- 触发：中（开始承载主题）
- 反转一：强（教化直接服务主题）
- 反转二：极强（普法直接服务主题）
- 收束：极强（升华主题）
```

### 步骤 2：写入 00_戏剧意图总纲.md

将导演解读输出写入：
```
{output_root}/{剧本名}/00_总览/00_戏剧意图总纲.md
```

### 步骤 3：M2 强制引用

M2 步骤 7（分镜大纲）必须引用 00_戏剧意图总纲.md 中的：
- 每镜戏剧意图（5 幕意图的细化）
- 角色情感轨迹（本镜的位置）
- 潜台词（本镜的潜台词分析）

## 五、与其他模块的集成

### 5.1 与 M_角色调用机制的集成

**新增触发时机**：导演解读阶段（M1.5）
