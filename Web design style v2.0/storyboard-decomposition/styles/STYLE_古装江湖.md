## 三、CUE 设计

### CUE①：开篇风格切换

**位置**：第 1 镜开篇

**操作**：
- 山水空镜 + 鼓声/笛声
- 色温跳变：2700K → 4500K
- 配乐：从无到渐起

**视觉效果**：
- 山水空镜建立古典感
- 鼓声/笛声渐起

**参考**：卧虎藏龙 / 英雄 / 刺客聂隐娘

### CUE②：中段决战升级

**位置**：第 9 镜前后

**操作**：
- 关键音效（剑鸣/鼓声）+ 0.3 秒静默
- 色温跳变：5000K → 4000K（自然 → 冷峻）
- 配乐：从轻到重 / 静默后鼓声爆发
- 光比：1:2 → 1:4

**视觉效果**：
- 从江湖到决战
- 冷化强化张力

**参考**：英雄 / 卧虎藏龙 / 一代宗师

### CUE③：升华

**位置**：最后一镜

**操作**：
- 色温跳变：4000K → 5500K（冷峻 → 暖金夕阳）
- 光比：1:4 → 1:∞（夕阳剪影）
- 笛声渐起

**视觉效果**：
- 夕阳剪影 + 笛声
- 江湖一骑红尘

**参考**：英雄 / 新龙门客栈 / 东邪西毒

## 六、BGM 风格

### 开篇（CUE①）

- 鼓声 + 笛声
- 慢节奏，建立古典感

### 中段（叙事）

- 古筝/琵琶 + 笛声
- 中等节奏

### CUE② 决战

- 关键音效（剑鸣）+ 0.3 秒静默
- 静默后鼓声爆发

### CUE③ 升华

- 笛声渐起
- 夕阳剪影
