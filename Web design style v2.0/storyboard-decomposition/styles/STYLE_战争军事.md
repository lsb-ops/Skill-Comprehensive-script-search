# STYLE — 战争军事

> 战争/军事/历史/抗战/谍战
> 战场史诗 + 灰黄调 + 国歌升华

---

## 一、风格定义

### 核心情绪
- 战场史诗（宏大、悲壮、热血）
- 灰黄调（黄沙、硝烟、战场）
- 爱国主义（牺牲、胜利、传承）
- 谍战紧张（潜伏、密码、接头）

### 视觉关键词
- 黄沙战场
- 硝烟弥漫
- 战壕
- 军装
- 国旗
- 战火

### 典型主题
- 抗战剧
- 谍战剧
- 解放战争
- 现代军事
- 历史战役
- 英雄牺牲

## 三、CUE 设计

### CUE①：开篇风格切换

**位置**：第 1 镜开篇

**操作**：
- 战场空镜 + 鼓声
- 色温跳变：4000K → 5000K
- 配乐：从无到鼓声渐起

**视觉效果**：
- 战场全景建立史诗感
- 鼓声渐起

**参考**：拯救大兵瑞恩 / 血战钢锯岭 / 长津湖

### CUE②：中段决战升级

**位置**：第 9 镜前后

**操作**：
- 关键音效（爆炸/呐喊）+ 0.3 秒静默
- 色温跳变：5000K → 4000K
- 配乐：从鼓声到弦乐爆发
- 光比：1:3 → 1:4

**视觉效果**：
- 从日常到决战
- 战火剪影

**参考**：拯救大兵瑞恩 / 血战钢锯岭 / 长津湖

### CUE③：升华（国旗剪影）

**位置**：最后一镜

**操作**：
- 色温跳变：4000K → 5500K（冷峻 → 暖金）
- 光比：1:∞（国旗剪影）
- 国歌/弦乐
- 慢动作

**视觉效果**：
- 国旗剪影
- 国歌弦乐
- 慢动作

**参考**：血战钢锯岭结尾 / 拯救大兵瑞恩结尾 / 长津湖

## 六、BGM 风格

### 开篇（CUE①）

- 鼓声
- 慢节奏，建立史诗感

### 中段（叙事）

- 鼓声 + 弦乐
- 中等节奏

### CUE② 决战

- 关键音效（爆炸）+ 0.3 秒静默
- 静默后弦乐爆发

### CUE③ 升华

- 国歌 / 弦乐
- 国旗剪影
