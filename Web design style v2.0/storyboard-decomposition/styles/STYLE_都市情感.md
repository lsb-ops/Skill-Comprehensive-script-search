# STYLE — 都市情感

> 都市/职场/爱情/家庭/离婚
> 现代都市感 + 情感细腻 + 暖光收束

---

## 一、风格定义

### 核心情绪
- 现代都市感（高楼大厦、玻璃幕墙、咖啡厅）
- 情感细腻（内心 OS、独白、回忆闪回）
- 现实主义（离婚、职场、家庭危机）
- 暖光收束（温馨、治愈、亲情/爱情救赎）

### 视觉关键词
- 城市天际线
- 玻璃幕墙反光
- 暖光（夕阳、咖啡厅灯光）
- 都市霓虹
- 雨天窗景
- 钢琴/吉他配乐

### 典型主题
- 都市爱情
- 职场奋斗
- 婚姻危机
- 离婚再恋
- 家庭矛盾
- 亲情救赎

## 二、视觉基线

### 色温曲线（5 幕结构）

| 幕 | 色温 | 色调 | 视觉 |
|---|---|---|---|
| 建制（幕1） | 4500K | 中性 | 自然基调 |
| 触发（幕2） | 5000K | 自然 | 日常 |
| 反转一（幕3） | 4500K-5000K | 自然 | 危机浮现 |
| 反转二（幕4） | 4000K | 冷化 | 冲突/分离 |
| 收束（幕5） | 5500K | 暖光 | 救赎/温馨 |

### 光比曲线

| 幕 | 光比 | 视觉 |
|---|---|---|
| 建制（幕1） | 1:2 | 柔光 |
| 触发（幕2） | 1:2 | 柔光 |
| 反转一（幕3） | 1:2 | 柔光 |
| 反转二（幕4） | 1:3 | 中等对比 |
| 收束（幕5） | 1:2 | 暖光柔光 |

### 色调

- **主色调**：自然中性（少量青橙）
- **冷化时刻**：冷蓝（离婚/危机）
- **暖光收束**：暖金（救赎/温馨）

## 三、CUE 设计

### CUE①：开篇风格切换

**位置**：第 1 镜开篇

**操作**：
- 城市天际线空镜 + 钢琴前奏
- 色温跳变：2700K → 5000K（暖场 → 自然）
- 配乐：从无到渐起

**视觉效果**：
- 城市全景建立都市感
- 配乐渐起烘托氛围

**参考**：迷失东京 Lost in Translation / 爱在黎明破晓前 Before Sunrise / 都市情缘

### CUE②：中段情绪升级

**位置**：第 8-9 镜前后

**操作**：
- 关键音效（玻璃碎裂/雨声/关门声）+ 0.3 秒静默
- 色温跳变：5000K → 4000K（自然 → 冷化）
- 配乐：从轻到重 / 静默后弦乐起
- 光比：1:2 → 1:3

**视觉效果**：
- 从日常到危机
- 冷化强化情感张力

**参考**：蓝色情人节 Blue Valentine / 婚姻故事 Marriage Story / 迷失东京

### CUE③：收束（救赎/温馨）

**位置**：最后一镜

**操作**：
- 色温跳变：4000K → 5500K（冷化 → 暖光）
- 配乐：钢琴独奏 + 渐弱
- 暖光渐亮

**视觉效果**：
- 从冷化到暖光
- 拥抱特写 + 暖光渐亮
- 温馨治愈

**参考**：爱在日落黄昏时 / 暖暖内含光 / 海街日记

## 六、BGM 风格

### 开篇（CUE①）

- 钢琴前奏 / 弦乐
- 慢节奏，建立都市感

### 中段（叙事）

- 中等节奏钢琴 + 弦乐
- 烘托对话和情绪

### CUE② 情绪升级

- 关键音效 + 0.3 秒静默
- 静默后弦乐爆发

### CUE③ 救赎

- 钢琴独奏
- 暖光渐亮
- 渐弱定帧
