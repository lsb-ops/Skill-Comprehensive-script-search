## 三、CUE 设计

### CUE①：开篇风格切换

**位置**：第 1 镜开篇

**操作**：
- 夜景空镜 + 低频音效
- 色温跳变：3500K → 5500K（暖场 → 冷蓝）
- 配乐：从无到低频渐起

**视觉效果**：
- 夜景建立悬疑感
- 低频音效

**参考**：七宗罪 / 沉默的羔羊 / 记忆碎片

### CUE②：中段真相升级

**位置**：第 9 镜前后

**操作**：
- 关键音效（心跳声/警笛声）+ 0.3 秒静默
- 色温跳变：5500K → 4000K
- 配乐：从轻到重 / 静默后低频爆发
- 光比：1:3 → 1:4

**视觉效果**：
- 从叙事到真相逼近
- 强对比强化张力

**参考**：七宗罪 / 记忆碎片 / 禁闭岛

### CUE③：真相揭示

**位置**：最后一镜

**操作**：
- 色温跳变：4000K → 6500K（冷峻 → 冷蓝）
- 配乐：静音 + 最后一击
- 真相瞬间揭示

**视觉效果**：
- 真相瞬间揭示
- 冷蓝剪影
- 戏剧性静默

**参考**：七宗罪结局 / 记忆碎片结局 / 致命魔术

## 六、BGM 风格

### 开篇（CUE①）

- 低频音效
- 慢节奏，建立悬疑感

### 中段（叙事）

- 中等节奏弦乐
- 持续紧张感

### CUE② 真相升级

- 关键音效（心跳声）+ 0.3 秒静默
- 静默后低频爆发

### CUE③ 真相揭示

- 静音 + 最后一击
- 戏剧性静默
