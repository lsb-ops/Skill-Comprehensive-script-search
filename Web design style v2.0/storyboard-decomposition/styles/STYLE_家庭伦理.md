## 三、CUE 设计

### CUE①：开篇风格切换

**位置**：第 1 镜开篇

**操作**：
- 家庭空镜 + 钢琴前奏
- 色温跳变：3000K → 5000K
- 配乐：从无到钢琴渐起

**视觉效果**：
- 家庭客厅建立温馨感
- 钢琴渐起

**参考**：步履不停 / 比海更深 / 海街日记

### CUE②：中段冲突升级

**位置**：第 9 镜前后

**操作**：
- 关键音效（关门声/争吵）+ 0.3 秒静默
- 色温跳变：5000K → 4000K（自然 → 冷化）
- 配乐：从轻到重 / 静默后弦乐起
- 光比：1:2 → 1:3

**视觉效果**：
- 从温馨到冲突
- 冷化强化张力

**参考**：步履不停 / 比海更深 / 东京家族

### CUE③：团圆和解

**位置**：最后一镜

**操作**：
- 色温跳变：4000K → 5500K（冷化 → 暖光）
- 配乐：钢琴独奏 + 渐弱
- 拥抱特写 + 暖光渐亮

**视觉效果**：
- 从冷化到暖光
- 拥抱特写 + 暖光渐亮
- 团圆治愈

**参考**：步履不停结尾 / 东京家族 / 海街日记

## 六、BGM 风格

### 开篇（CUE①）

- 钢琴前奏
- 慢节奏，建立温馨感

### 中段（叙事）

- 钢琴 + 弦乐
- 中等节奏

### CUE② 冲突升级

- 关键音效（关门声）+ 0.3 秒静默
- 静默后弦乐起

### CUE③ 团圆

- 钢琴独奏
- 暖光渐亮
- 渐弱定帧
