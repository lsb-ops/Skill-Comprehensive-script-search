## 三、CUE 设计

### CUE①：开篇风格切换（前 0-10 秒）

**位置**：第 1 镜开篇

**操作**：
- 色温跳变：2700K → 4500K（暖场 → 自然）
- 配乐：从无到渐起（钢琴/弦乐）
- 光比：1:1 → 1:2

**视觉效果**：
- 从暖场建立日常
- 镜头运动引入人物
- 配乐渐起烘托氛围

**参考**：教父开篇黑屏 + 配乐 / 上海滩许文强登场

### CUE②：中段教化升级（30-40 秒处）

**位置**：第 9 镜前后

**操作**：
- 关键音效 + 0.3 秒静默（如刹车声、脚步声停止）
- 色温跳变：5000K → 4000K（自然 → 冷静）
- 配乐：从轻到重 / 静默 0.3 秒后弦乐起
- 光比：1:2 → 1:3

**视觉效果**：
- 从日常升级到严肃
- 主光变冷
- 配乐爆发

**参考**：教父谈判桌 / 功夫包租公被敲懵 / 低俗小说教化者走开

### CUE③：升华（70-80 秒处）

**位置**：最后一镜（背影收束）

**操作**：
- 色温跳变：4000K → 5500K（冷静 → 暖金）
- 光比：1:3 → 1:∞（剪影）
- Halation：开
- AI 后期逆光晕强化
- 配乐高峰 + 内心 OS

**视觉效果**：
- 从教化到胜利
- 逆光剪影 + 暖金光环
- 背影渐远
- 普法先锋的光环美学

**参考**：教父背影收束 / 阿拉伯的劳伦斯逆光剪影 / 天堂电影院结尾升华 / 功夫如来神掌光环

## 六、BGM 风格

### 开篇（CUE①）

- 钢琴前奏 / 弦乐渐起
- 慢节奏，建立高级感

### 中段（叙事）

- 中等节奏钢琴 + 弦乐
- 烘托对话和情绪

### CUE② 教化升级

- 关键音效（刹车声）+ 0.3 秒静默
- 静默后弦乐爆发

### CUE③ 升华

- 配乐高峰（-6dB）
- 内心 OS 起
- 配乐延续 + 渐弱定帧

### 音量曲线

| 阶段 | 音量 |
|---|---|
| 静默 | -∞ |
| 环境音 | -40 dB |
| 配乐渐起 | -30 dB |
| 配乐中等 | -20 dB |
| 配乐高潮 | -10 dB |
| 配乐高峰（CUE③） | -6 dB |

## 八、典型对白风格

### 主角台词
- 简短有力
- 教化性
- 隐含道德判断
- 例：「这届助理真难带，一点媒介素养都没有。」（内心 OS）

### 对手台词
- 嚣张 → 被教化
- 转变过程清晰
- 例：「你就是刘总？」「让开。」

### 教化台词（可选）
- 普及法律/道德
- 简短直接
- 例：「非法获取公民个人信息，最高可判处七年有期徒刑。」
