## 三、CUE 设计

### CUE①：开篇风格切换

**位置**：第 1 镜开篇

**操作**：
- 太空/城市空镜 + 电子音效
- 数字故障特效
- 色温跳变：8000K → 5500K

**视觉效果**：
- 太空/赛博城市建立未来感
- 电子音效渐起

**参考**：银翼杀手 2049 / 黑客帝国 / 盗梦空间

### CUE②：中段决战升级

**位置**：第 9 镜前后

**操作**：
- 关键音效（电子脉冲/爆炸）+ 0.3 秒静默
- 数字故障特效强化
- 色温跳变：5500K → 4000K
- 光比：1:2 → 1:4

**视觉效果**：
- 从日常到决战
- 数字故障强化张力

**参考**：银翼杀手 / 黑客帝国 / 头号玩家

### CUE③：升华

**位置**：最后一镜

**操作**：
- 光速推进 + 静音
- 色温跳变：4000K → 6500K
- 霓虹剪影

**视觉效果**：
- 光速推进
- 霓虹剪影 + 静音

**参考**：2001 太空漫游 / 星际穿越 / 银翼杀手结尾
