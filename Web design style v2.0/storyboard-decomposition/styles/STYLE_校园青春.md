## 三、CUE 设计

### CUE①：开篇风格切换

**位置**：第 1 镜开篇

**操作**：
- 阳光教室空镜 + 吉他前奏
- 色温跳变：3500K → 5000K
- 配乐：从无到渐起

**视觉效果**：
- 阳光教室建立青春感
- 吉他渐起

**参考**：怦然心动 / 那些年我们一起追的女孩 / 致我们终将逝去的青春

### CUE②：中段情绪升级

**位置**：第 9 镜前后

**操作**：
- 关键音效（铃声/呐喊）+ 0.3 秒静默
- 色温跳变：5000K → 4500K
- 配乐：从轻到重 / 静默后弦乐起
- 光比：1:2 → 1:3

**视觉效果**：
- 从日常到危机
- 微冷化

**参考**：那些年 / 我的少女时代 / 致青春

### CUE③：升华（黄昏剪影）

**位置**：最后一镜

**操作**：
- 色温跳变：4500K → 5500K
- 光比：1:3 → 1:∞（黄昏剪影）
- 吉他 + 弦乐
- 慢动作

**视觉效果**：
- 黄昏剪影
- 慢动作 + 暖金
- 青春告别

**参考**：那些年 / 我的少女时代 / 七月与安生

## 六、BGM 风格

### 开篇（CUE①）

- 吉他前奏
- 慢节奏，建立清新感

### 中段（叙事）

- 吉他 + 钢琴
- 中等节奏

### CUE② 情绪升级

- 关键音效（铃声）+ 0.3 秒静默
- 静默后弦乐起

### CUE③ 升华

- 吉他独奏 + 弦乐
- 黄昏剪影 + 慢动作
