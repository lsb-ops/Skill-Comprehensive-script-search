# 30 分钟剧本→分镜教程

> **目标**：看完能跑通，不讲架构，只讲最短路径
> **时间**：30 分钟
> **前置**：macOS / Python 3.11+ / 1 个 txt 剧本

---

## 5 步跑通

### Step 1：准备剧本（5 分钟）

把你的剧本存为 `my_script.md`，格式建议：

```markdown
# 我的剧本标题

> 总时长 60 秒 ｜ 镜数 6

## 第一幕：建制

### 镜头 001｜场景描述
- 景别：MS 中景
- 焦距：50mm
- 运镜：固定
- 时长：5 秒
- 人物：角色 A
- 动作：动作描述
- 口播：「对白内容」

### 镜头 002｜...
```

### Step 2：跑 M1-M4 启发式（5 分钟）

```bash
cd /Users/lin/Desktop/霸道总裁/剧本
./skill/scripts/run_skill.sh my_script.md --output-dir ./output --total-duration 60
```

**会得到**：
- `output/m1/m1_output.json` — 剧本结构
- `output/m2/m2_output.json` — 分镜大纲
- `output/m3/镜*.txt` — **每镜的 v3.5 治本提示词**（这就是你的核心交付物）
- `output/m4/continuity_report.json` — 跨镜一致性报告

**没得到视频文件**（这是正常的 — skill 暂只生成提示词，不生成视频）

### Step 3：复制提示词到视频 API（10 分钟）

打开 `output/m3/镜001_xxx.txt`，**整段复制**到即梦/可灵网页：

- 即梦：https://jimeng.jianying.com
- 可灵：https://klingai.com

按 API 提示粘贴 + 选参数（9:16 / 5 秒）→ 生成视频

### Step 4：（可选）跨集资产继承（5 分钟）

如果你要做 EP02（继承 EP01 角色/场景/道具）：

```bash
cd /Users/lin/Desktop/霸道总裁/剧本/skill
python3 scripts/m10_inherit.py ../output/m3/ ../output/m3_ep02/ --ep EP02 \
  --characters-dir ./character_lib \
  --scenes-dir ./scene_lib \
  --props-dir ./prop_lib
```

**会得到**：`output/m3_ep02/镜*.txt` — 自动注入 EP01 资产延续块（必须保留特征 + prompt 模板）

**前置条件**：
1. `character_lib/人物_xxx/character.json` 里 `ep_appearances.EP02` 字段已配置
2. 提示词 txt 里有 `【人物_xxx：】` 标记

### Step 5：（可选）质量验证（2 分钟）

```bash
cd /Users/lin/Desktop/霸道总裁/剧本/skill
python3 scripts/ai_verify.py ../output/m3/ --lib-dir ./ --threshold 0.7
```

**会得到**：每镜 5 维评分（0-1），< 0.7 标记 needs_redo

**5 维**：
- 文本一致性（40%）：是否含 `must_keep` 特征
- 无元注释（15%）：无"决策："/"修复："段首
- 段落完整（15%）：6 段齐全
- 资产引用（15%）：含【人物_xxx：】等
- 工程关键词（15%）：T-stop/K 色温/焦距

## 常见问题

### Q1：跑出来不是 6 镜，是 2 镜？
A：M1 启发式对短剧本/格式不规范的剧本不友好。
**解决**：手写 5 镜格式（参考 examples/example-古早霸总-EP02/EP02-简版5镜.md）

### Q2：m10 注入显示 0 个资产？
A：M3 txt 里没有 `【人物_xxx：】` 标记。
**解决**：先确保 M1/M2/M3 跑通，提示词里有人物引用。

### Q3：M7 评分 < 0.7 怎么办？
A：先看哪个维度低：
- 文本一致性低 → 跑 m10 注入 must_keep
- 段落不完整 → 重新跑 M3
- 资产引用低 → 补【人物_xxx：】标记

### Q4：想做 LLM 增强模式？
A：需要 `ANTHROPIC_API_KEY` 环境变量：
```bash
export ANTHROPIC_API_KEY=sk-xxx
./run_skill.sh my_script.md --llm
```
