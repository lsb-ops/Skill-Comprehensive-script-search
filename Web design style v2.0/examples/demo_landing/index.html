常见问题

还有疑问?

Flow 和 Todoist / Things 有什么不同?

Todoist 是清单工具, Flow 是操作系统。我们用 AI 接管"分类"和"决策", 你只需要做一件事: 进入专注模式。

数据隐私如何保障?

所有任务数据端到端加密, 我们不读取你的内容, AI 分类在本地 LLaMA 模型优先, 云端 GPT-4 仅在你显式开启时使用。

可以导入现有数据吗?

支持从 Todoist / Things / TickTick / Apple 提醒事项一键导入, 首次登录时会有引导流程。

免费版够用吗?

如果你每月任务不超过 50 个, 免费版完全够用。 我们不希望你为了基本功能付费。
