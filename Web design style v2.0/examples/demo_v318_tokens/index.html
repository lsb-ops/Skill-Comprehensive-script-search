v3.18 Design Tokens

基于 Radix UI / shadcn / Tailwind v4 / Open Props / MD3 / Apple HIG 的综合研究, v3.18 落地:

  • OKLCH + HSL 拆段 (shadcn 风格)
  • 语义色 success / warning / error / info
  • 5 级阴影 (Open Props 风格)
  • MD3 12 级 duration
  • Apple spring 三参数 (mass/stiffness/damping)
  • CVD 色盲安全 8 色 (Okabe-Ito)
  • WCAG 2.2 focus-visible 2px / 3:1 对比
