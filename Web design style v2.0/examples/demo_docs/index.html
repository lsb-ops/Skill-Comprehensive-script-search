Tasks API

Tasks 是 Flow 的核心资源, 表示用户的一个待办事项。 通过 Tasks API 你可以创建、查询、更新、删除任务, 以及批量操作。

概览

每个 Task 包含以下字段:

{
  "id": "tsk_01HXYZ...",
  "title": "完成 Q2 OKR 评审",
  "description": "准备 5 张幻灯片, 邀请 8 人",
  "priority": "high",
  "status": "todo",
  "due_at": "2026-06-30T17:00:00Z",
  "project_id": "prj_01H...",
  "labels": ["urgent", "review"],
  "ai_confidence": 0.94,
  "created_at": "2026-06-25T08:30:00Z",
  "updated_at": "2026-06-26T03:15:00Z"
}
// 创建任务
const task = await fetch('https://api.flow.app/v1/tasks', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer ' + API_KEY,
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    title: '完成 Q2 OKR 评审',
    priority: 'high',
    due_at: '2026-06-30T17:00:00Z'
  })
}).then(r => r.json());

console.log(task.id); // tsk_01HXYZ...
id
string
任务唯一 ID, 格式 tsk_ 开头
title
string
任务标题, 1-200 字符
priority
enum
low / medium / high / urgent
ai_confidence
float
AI 自动分类置信度 (0-1), ≥0.8 视为可靠

端点

所有 Tasks 端点都在 https://api.flow.app/v1/tasks 下。

列出任务

200 OK

创建任务

POST /v1/tasks

请求体同上方 字段说明。

更新任务

PATCH /v1/tasks/{id}

只需传要修改的字段。

删除任务

DELETE /v1/tasks/{id}

返回 204 No Content。

错误处理

Flow API 使用标准 HTTP 状态码, 详细错误信息在响应体的 error 字段中。

  • 400 invalid_request — 参数缺失或格式错误
  • 401 invalid_api_key — API key 无效
  • 403 forbidden — 无权访问该资源
  • 404 not_found — 任务不存在
  • 429 rate_limit_exceeded — 触发速率限制
  • 500 internal_error — 服务异常, 可安全重试
  • 503 service_unavailable — 服务暂时不可用

建议使用指数退避:

async function withRetry(fn, maxRetries = 3) {
  for (let i = 0; i < maxRetries; i++) {
    try { return await fn(); }
    catch (e) {
      if (!e.retryable || i === maxRetries - 1) throw e;
      await new Promise(r =>
        setTimeout(r, Math.pow(2, i) * 1000)
      );
    }
  }
}
