```

镜头2: 特写他的表情，眉头紧皱，嘴唇颤抖。
```

## 镜头角度（Camera Angles）

| 角度 | 效果 |
|------|------|
| 平视 | 中性自然 |
| 俯视 | 显得弱小、脆弱 |
| 仰视 | 显得强大、有权威 |
| 鸟瞰 | 上帝视角、地理建立 |
| 荷兰角 | 不安、紧张、焦虑 |

## 连续性规则（核心）

### 180度规则
两个角色对话，相机必须在轴线一侧，不能跨越。

### 动作匹配
切到同一动作的不同角度，动作必须无缝衔接。

### 视线匹配
角色看向某物，下一个镜头展示他看的内容。

## 表情优化词库

**基础表情：** 平静、微笑、皱眉、惊讶、悲伤、愤怒、思考

**复合表情：**
- 似笑非笑（压抑）
- 欲言又止（压抑）
- 眼神闪烁（不安/兴奋）
- 嘴角抽搐（压抑）
- 喉结滚动（紧张）
- 眉心拧成川字（焦虑）
- 空洞眼神（绝望/麻木）
- 眉头紧锁（愤怒/焦虑）

**修饰词：** 细微的、隐约的、明显的、瞬间的、持续的

---

## 动作优化框架

**三要素：** 主体（谁）+ 内容（做什么）+ 目的（为什么）

**优化方向：**
- 模糊 → 具体：`走` → `步伐沉重地走向`
- 单一 → 复合：加入眼神，呼吸、姿态
- 静态 → 动态：加入力量、节奏、方向感

**动作词库：**
- 行走：踱步、蹒跚、疾走、缓行、踉跄
- 手势：攥紧、摊开、颤抖、挥动、束缚
- 姿态：挺直、佝偻、僵硬、颤抖、被束缚
