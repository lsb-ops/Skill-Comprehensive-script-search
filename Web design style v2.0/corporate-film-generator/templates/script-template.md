## 叙事结构

### 三幕式布局

```
┌─────────────────────────────────────────────────────┐
│                    第一幕：建置 (25-30%)              │
│  序章({intro}s) + 第一章({ch1}s)                    │
│  AIAEST: Attention → Interest                        │
│  核心任务: 建立主角，制造好奇                         │
└─────────────────────────────────────────────────────┘
                           ↓
┌─────────────────────────────────────────────────────┐
│                  第二幕：对抗 (50-60%)                │
│  第二章({ch2}s)                                     │
│  AIAEST: Action → Emotion                            │
│  核心任务: 展示能力，情感共鸣                         │
└─────────────────────────────────────────────────────┘
                           ↓
┌─────────────────────────────────────────────────────┐
│                   第三幕：解决 (20-25%)              │
│  第三章({ch3}s) + 终章({outro}s)                    │
│  AIAEST: Emotion → Satisfaction                      │
│  核心任务: 价值确认，行动号召                         │
└─────────────────────────────────────────────────────┘
```

### AIAEST映射

| 章节 | 时长 | AIAEST | 情绪目标 | 核心任务 |
|------|------|--------|----------|----------|
| 序章 | {intro}s | A | 好奇→兴趣 | 开场钩子，悬念设置 |
| 第一章 | {ch1}s | I | 兴趣→认同 | 痛点共鸣，价值展示 |
| 第二章 | {ch2}s | A-E | 认同→震撼 | 行动展示，能力证明 |
| 第三章 | {ch3}s | E | 震撼→信任 | 情感故事，高光时刻 |
| 终章 | {outro}s | S | 信任→行动 | 信任背书，行动号召 |

```

    │    ╰────╯────╰────╯────╰────╯
    └──────────────────────────────→ 时间
       A     I     A     E     S
      (注意)(兴趣)(行动)(情感)(满足)

   序章            第一章            第二章            第三章           终章
```

## 版本变体

### 情感优先版

**差异化：** 减少数据，增加故事和情感叙事

### 数据优先版

**差异化：** 增加业绩数据，减少情感叙事

### 1分钟精华版

**结构：**
```
0-5s:   开场hook（Attention）
5-25s:  核心价值（Interest+Action）
25-45s: 实力证明（Emotion）
45-55s: 信任背书
55-60s: 行动号召（Satisfaction）
```

## 质量自检清单

```
叙事完整性：
[ ] 有一条清晰的主线贯穿全片
[ ] 开场制造了好奇或悬念
[ ] 有至少一个情感共鸣点
[ ] 高潮部分展示了独特差异化
[ ] 结尾有明确的行动号召

差异化验证：
[ ] 明确说明"为什么选你"
[ ] 差异化有证据/数据支撑
[ ] 差异化可感知（非自说自话）
[ ] 有竞争对手做不到的事

信息质量：
[ ] 核心信息不超过3个
[ ] 避免信息堆砌
[ ] 数据有具体来源
[ ] 情感曲线渐进而非突变

敏感信息：
[ ] 无未经授权的第三方品牌
[ ] 无虚假或无法核实的声明
[ ] 无侵权内容
[ ] 敏感话题已妥善处理
```
