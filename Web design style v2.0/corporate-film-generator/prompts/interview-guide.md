# 企业宣传片追问话术指南 v3.0

> 全球方法论融合 + 自适应问题调整 + AIAEST叙事映射

---

## 核心框架

### 1. AIAEST叙事映射

| 阶段 | 情感目标 | 追问重点 |
|------|----------|----------|
| **A**ttention | 好奇/兴趣 | 开场钩子、悬念设计 |
| **I**nterest | 投入/好奇 | 痛点共鸣、价值展示 |
| **A**ction | 期待/认同 | 行动展示、能力证明 |
| **E**motion | 震撼/感动 | 情感故事、高光时刻 |
| **S**atisfaction | 信任/行动 | 信任背书、行动号召 |

### 2. 自适应问题调整

**回答质量评估 → 动态追问：**

```
用户回答
    │
    ├─ 模糊/简短 → 触发【具体化追问】
    │    "能具体说说吗？比如..."
    │
    ├─ 具体/丰富 → 触发【深挖追问】
    │    "这个让我很感兴趣，能展开讲讲吗？"
    │
    ├─ 跳跃/发散 → 触发【聚焦追问】
    │    "您提到的X很有价值，回到我刚才的问题..."
    │
    └─ 沉默/跳过 → 记录待补充，继续下一问
```

### 3. 追问深度层级

| 层级 | 适用场景 | 追问句式 |
|------|----------|----------|
| L1-事实层 | 回答笼统 | "能具体说说吗？" |
| L2-原因层 | 回答表面 | "为什么这么想？" |
| L3-意义层 | 回答浅显 | "这意味着什么？" |
| L4-例外层 | 回答确定 | "有没有例外情况？" |
| L5-假设层 | 回答完成 | "如果没有...会怎样？" |

## Wave 1：企业介绍 + 产品服务

### 企业介绍（A-I：建立好奇）

#### 开场钩子问题

**Q1 - 一句话介绍：**
> "如果用一句话向一个完全不了解你们行业的人介绍公司，你会怎么说？"

**追问策略：**
- 回答过短 → "能再多说一点吗？比如你们做什么、解决什么问题？"
- 回答具体 → "很好！这个'XX'是你们最核心的价值，对吗？"

**Q2 - 竞争优势：**
> "在客户眼中，贵公司与其他同行最大的不同是什么？"

**追问策略（苏格拉底法）：**
- "什么让你认为这是'最大的不同'？"
- "客户也这么认为吗？有具体反馈吗？"
- "这个优势5年后还会存在吗？"

**Q3 - 行业地位：**
> "在行业中，贵公司处于什么位置？有没有具体数据？"

**追问策略：**
- "这个地位是怎么获得的？花了多长时间？"
- "三年后您想处于什么位置？"

### 产品服务（I-A：展示能力）

#### 痛点挖掘

**Q1 - 客户痛点：**
> "客户在遇到什么问题时会第一时间想到你们？"

**追问策略（心理透镜）：**
- "客户通常用什么词描述这个问题？"
- "这个问题如果不解决，客户会有什么后果？"
- "是'必须解决'还是'有了更好'？"

**Q2 - 解决方案：**
> "你们具体是怎么解决这个问题的？"

**追问策略：**
- "和其他解决方案相比，你们的不同在哪里？"
- "有没有什么技术是只有你们掌握的？"

**Q3 - 不可替代性：**
> "客户为什么会持续选择你们而不是换供应商？"

**追问策略（战略透镜）：**
- "有没有客户是因为你们而离不开的？"
- "你们做了什么让客户'换不了'？"

#### STAR法则：产品成功故事

> "能讲一个产品帮助客户成功的具体故事吗？用STAR法则："

**S（情境）**: "当时客户面临什么情况？"
**T（任务）**: "他们想要达成什么？"
**A（行动）**: "你们的产品具体是怎么帮助他们的？"
**R（结果）**: "最终效果如何？有数据吗？"

#### 标签收集

```markdown
<!-- 产品线: -->
<!-- 核心产品: -->
<!-- 解决痛点: -->
<!-- 差异化: -->
<!-- 技术壁垒: -->
<!-- 专利: -->
```

## Wave 2：团队实力 + 发展历程

### 团队实力（E：情感共鸣）

#### 创始人故事

**Q1 - 创业动机：**
> "创始人为什么选择做这件事？最初的梦想是什么？"

**追问策略（STAR法则）：**

**S（情境）**: "当时是什么情况促使这个决定？"
**T（任务）**: "面临的最大挑战是什么？"
**A（行动）**: "具体采取了什么行动？"
**R（结果）**: "结果如何？现在回头看怎么评价？"

**Q2 - 团队精神：**
> "如果用三个词来形容公司的团队氛围，会是哪三个？"

**追问策略：**
- "有没有一个具体的场景或故事能体现这种氛围？"
- "这种氛围是怎么形成的？"

**Q3 - 关键人才：**
> "团队中有没有什么特别厉害的人？他们的加入带来了什么改变？"

**追问策略：**
- "为什么这些人才选择你们而不是大公司？"
- "他们最看重的是什么？"

### 发展历程（E-S：证明实力）

#### 高光时刻

**Q1 - 里程碑：**
> "回顾企业发展历程，哪个时刻最让你感到骄傲？"

**追问策略（STAR法则）：**

**S（情境）**: "当时是什么情况？"
**T（任务）**: "面临什么困难或挑战？"
**A（行动）**: "怎么克服的？做了哪些关键决定？"
**R（结果）**: "带来了什么改变？有具体数字吗？"

**Q2 - 至暗时刻：**
> "有没有遇到过差点过不去的坎？最后怎么过来的？"

**追问策略（心理透镜）：**
- "当时最难的是什么？"
- "这个经历让团队学到了什么？"
- "这段经历现在还影响着什么决策？"

- "什么情况下会产生这个结果？"
- "如何预防或避免？"
- "早期预警信号是什么？"

## Wave 4：受众分析 + 竞争分析

### 受众分析（A-E：理解共鸣）

#### 受众画像

**Q1 - 目标受众：**
> "这部宣传片主要是给谁看的？他们的年龄、职位、关注点是什么？"

**追问策略：**
- "是'拍板的人'还是'影响决策的人'？"
- "谁会主动搜索你们？谁是被动看到的？"

**Q2 - 深层需求：**
> "这些受众最关心什么？最担心什么？"

**追问策略（心理透镜）：**
- "他们真正关心的是什么？说的和想的一样吗？"
- "他们对企业的信任建立在什么基础上？"

#### 情感设计

**Q3 - 情感目标：**
> "观众看完宣传片后，您希望他们心里想的是什么？"

**追问策略：**
- "是'震撼'、'信任'、'亲切'、还是'敬佩'？"
- "看完后最希望他们记住的一件事是什么？"

**Q4 - 行动号召：**
> "您希望观众看完后做什么？"

**追问策略：**
- "是直接联系？购买？推荐？还是只是'记住了'？"
- "如何衡量宣传片的效果？"

### 竞争分析（E：建立独特性）

#### 竞争定位

**Q1 - 竞争对手：**
> "如果只能选一个最强的竞争对手，是谁？"

**追问策略：**
- "为什么是他们而不是别人？"
- "他们最近在做什么？"

**Q2 - 核心差异：**
> "客户选择你们而不是他们的理由是什么？"

**追问策略（战略透镜）：**
- "这个差异是他们承认的还是你们自己认为的？"
- "有没有客户同时用过你们和竞争对手？怎么评价？"

**Q3 - 竞争壁垒：**
> "有什么护城河是对手短期难以复制的？"

**追问策略：**
- "这个壁垒5年后还在吗？"
- "如果对手投入同样资源，能追上吗？"

#### 战略透镜追问

**负面空间：**
> "我们有什么还没说到、但其实很重要的潜在优势？"

**机会成本：**
> "我们选择做X，意味着放弃了什么？"

## Wave 5：风险识别 + 叙事设计

### 风险识别（S：信任保障）

#### 敏感话题

**Q1 - 敏感信息：**
> "有什么话题或信息，最好不要在宣传片中提及？"

**追问策略：**
- "有没有诉讼、负面新闻、争议话题？"
- "有没有不能提的竞争对手或合作伙伴？"

> "在品牌规范或法律合规方面，有什么必须遵守的要求？"

**追问策略：**
- "有没有必须包含或不能包含的内容？"

**追问策略：**

### 叙事设计（整合所有元素）

#### 企业精神

**Q1 - 企业灵魂：**
> "如果用一句话概括企业的灵魂或精神，会是什么？"

**追问策略（心理透镜）：**
- "您说的是'我们想成为的'，还是'我们本来就是的'？"
- "如果用三个词来形容，会是哪三个？"

> "有没有一个故事，最能代表这家公司？"

**追问策略（STAR法则）：**

## 追问技巧汇总

### 处理不同回答质量

| 回答类型 | 识别特征 | 追问策略 |
|----------|----------|----------|
| 模糊简短 | "还不错"、"挺好的" | 具体化：给选项或举例 |
| 具体丰富 | 有细节有数字 | 深挖：追问原因/意义/例外 |
| 跳跃发散 | 说到其他话题 | 聚焦：承认价值，拉回主题 |
| 确定绝对 | "一定"、"绝对" | 挑战：有没有例外？ |
| 沉默跳过 | 不回答或换话题 | 记录：待补充，继续下一问 |

### 追问深度递进

```
第1轮：问事实（是什么）
↓ 能具体说说吗？
第2轮：问原因（为什么）
↓ 为什么这么想？
第3轮：问意义（意味着什么）
↓ 这对客户意味着什么？
第4轮：问例外（有没有相反情况？）
↓ 有没有客户不这么认为的？
第5轮：问假设（如果没有...？）
↓ 如果没有这个优势会怎样？
```

### 矛盾信息处理

**策略：**

1. 指出矛盾："我注意到您说X，但之前说Y..."
2. 请用户澄清："是我理解错了，还是情况有变化？"
3. 记录多版本："那我们先保留两种说法，后续确认"

### 敏感话题处理

| 情况 | 话术 |
|------|------|
| 需要敏感数据 | "如果方便可以说营收规模，不方便我们用'行业平均'来描述" |
| 负面历史 | "感谢坦诚，这段经历带来了什么改变？" |
| 竞争对手 | "提到竞争对手，您觉得他们会怎么评价你们？" |

## 对话示例

### 完整对话示例

**AI**: "请用一句话向完全不了解你们行业的人介绍公司。"

**用户**: "我们是一家做工业互联网的公司。"

**AI**: "工业互联网，帮助企业做设备联网和数据处理，对吗？那我很好奇——在客户眼中，你们和其他工业互联网公司最大的不同是什么？"

**用户**: "主要是数据处理能力，我们有自己的算法。"

**AI**: "自己的算法——这很有意思。什么让你认为这是最大的不同？是客户这么反馈的，还是你们自己的判断？"

**用户**: "都有。客户经常说我们的数据分析比别家准很多。"

**AI**: "客户反馈是很好的证明。能给我一个具体的例子吗？比如哪个客户、什么场景、数据准到什么程度？"

**用户**: "比如三一重工，我们帮他们优化了设备维护，故障率降了40%。"

**AI**: "故障率降了40%——这是非常具体的数据。用STAR法则来说说这个案例吧：当时三一面临什么情况？你们具体做了什么？"
