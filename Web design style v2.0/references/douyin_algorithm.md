## 一、抖音推荐算法核心指标

### 1.1 完播率 > 一切

**核心公式**:
```
推荐权重 = 完播率 × 0.4 + 互动率 × 0.3 + 关注转化 × 0.2 + 其他 × 0.1
```

**完播率定义**: 完整看完视频的用户数 / 播放数

**关键阈值**:
- 完播率 ≥ 60%：进入更大流量池
- 完播率 < 30%：限制推荐

### 1.2 互动率指标

| 指标 | 权重 | 提升方法 |
|------|------|----------|
| 评论率 | 高 | 钩子问题、争议观点 |
| 点赞率 | 中 | 情感共鸣 |
| 收藏率 | 中 | 干货知识 |
| 转发率 | 中 | 实用工具 |
| 关注转化率 | 中 | 持续价值承诺 |

### 1.3 流量池模型

```
初始池 (200-500 播放)
  ↓ 完播率 ≥ 50%
二级池 (1000-5000 播放)
  ↓ 完播率 ≥ 60% + 互动
三级池 (1万-10万 播放)
  ↓ 持续高互动
热门池 (百万+ 播放)
```

## 二、前 3 秒钩子学

### 2.1 为什么要前 3 秒？

抖音的"黄金 3 秒"决定**初始完播率**。算法通过前 3 秒判断：
- 用户是否会继续看
- 是否值得推荐给更多人

### 2.2 7 种钩子类型

#### ① 问题钩子

```
你知道吗？每天喝咖啡其实...
为什么年轻人越来越不愿意结婚？
```

**技巧**：用"你"开头，激发好奇心

#### ② 反差钩子

```
我以为学 Python 很难，结果...
以为减肥要节食，结果反而胖了
```

**技巧**：先"以为"再"结果"，制造反转

#### ③ 数字钩子

```
5 个真相，第 3 个扎心了
90 天，我学到了这 3 件事
```

**技巧**：数字暗示有干货

#### ④ 视觉钩子

```
[大字报：48px 红字]
"AI 不会取代你"
```

**技巧**：视觉冲击力抢注意力

#### ⑤ 悬念钩子

```
我做了 1 个决定，改变了一生
这件事，我后悔了 10 年
```

**技巧**：省略关键信息，逼用户看下去

#### ⑥ 利益钩子

```
看这一条视频，省你 100 小时
学会这招，副业月入 1 万
```

**技巧**：直接说收益

#### ⑦ 共鸣钩子

```
你也经常熬夜吗？
说真的，谁不想多赚点钱
```

**技巧**：唤起情感连接

## 三、节奏设计

### 3.1 节奏曲线

抖音视频应保持**前紧后松**的节奏：

```
强度
 ↑
 │      ╱╲
 │     ╱  ╲    ╱╲
 │    ╱    ╲  ╱  ╲
 │   ╱      ╲╱    ╲
 │  ╱              ╲___
 └─────────────────────→ 时间
   0-3s   中段    结尾
```

### 3.2 节奏原则

- **0-3s**：紧张，钩子密度高
- **3-15s**：平稳，信息密度高
- **15-25s**：渐强，情绪转折
- **25-30s**：松弛，CTA 引导

### 3.3 反模式

| 反模式 | 后果 |
|--------|------|
| 全程紧绷 | 用户疲劳，划走 |
| 全程松弛 | 没吸引力，划走 |
| 中间断层 | 用户失去主线 |
| 结尾平淡 | 没有 CTA，转化低 |

## 四、文案优化

### 4.1 标题（≤ 22 字）

**5 大公式**:

1. **数字 + 关键词 + 钩子**
   - `5 个真相，第 3 个扎心了`
2. **如何/怎么 + 收益**
   - `如何 30 天学会 Python？`
3. **你以为 + 反转**
   - `你以为省钱，其实亏更多`
4. **我不允许 + 对象**
   - `我不允许你还不知道 XX`
5. **省略号 + 悬念**
   - `90 天后，我变成了...`

### 4.2 正文（80-150 字）

**结构**:
```
[钩子句] 30 字以内
[展开] 3-4 句话
[CTA] 引导评论/关注
```

**节奏**:
- 短句优先（≤ 15 字）
- 用 `。` 切分节奏
- 偶尔用 `？` `！` 增加情绪

### 4.3 Hashtag 策略

| 类型 | 数量 | 示例 |
|------|------|------|
| 大流量 | 1-2 | `#知识分享` `#干货` |
| 垂直领域 | 2-3 | `#AI写作` `#职场干货` |
| 趋势话题 | 1-2 | `#2026爆款` |
| 自创话题 | 0-1 | `#我的XX日记` |

**总数量**: 5-8 个最佳

## 五、发布时间

### 5.1 黄金时段

| 时段 | 流量 | 适用 |
|------|------|------|
| 7:00-9:00 | 高 | 通勤、学习 |
| 12:00-14:00 | 高 | 午休 |
| 18:00-20:00 | 极高 | 下班 |
| 21:00-23:00 | 极高 | 睡前 |

### 5.2 工作日 vs 周末

- **工作日**: 知识类、工具类更好
- **周末**: 娱乐类、生活类更好

## 六、视频长度

### 6.1 长度与完播率

| 时长 | 完播率 | 适用 |
|------|--------|------|
| 15s | 高 | 短平快、钩子 |
| 30s | 中 | 常规内容 |
| 60s | 中 | 深度内容 |
| >90s | 低 | 除非非常吸引 |

### 6.2 推荐长度

- **新手**: 15-30 秒（完播率优先）
- **进阶**: 30-60 秒（深度内容）
- **剧情/故事**: 60-90 秒（连续性）

## 七、互动引导（CTA）

### 7.1 CTA 类型

| 类型 | 示例 | 适用 |
|------|------|------|
| 评论引导 | "评论区告诉我..." | 所有 |
| 关注引导 | "关注我，下期讲..." | 系列内容 |
| 收藏引导 | "收藏起来慢慢看..." | 干货 |
| 分享引导 | "@你的朋友一起..." | 实用工具 |
| 点击引导 | "点击主页看更多..." | 主页有货 |

### 7.2 CTA 时机

- **结尾 2-3 秒**：最常见
- **中段穿插**：不打断主线
- **全程隐性**："我后面会讲 X，关注我不错过"

## 八、违规与红线

### 8.1 内容红线

| 红线 | 后果 |
|------|------|
| 政治敏感 | 封号 |
| 广告法禁用词 | 限流 |
| 虚假宣传 | 处罚 |
| 暴力色情 | 封号 |
| 版权侵权 | 下架 |

### 8.2 广告法禁用词

| 类别 | 禁用词 |
|------|--------|
| 最 | 最好、最强、最便宜 |
| 第一 | 第一、唯一、首个 |
| 绝对 | 绝对、肯定、100% |
| 权威 | 国家、央视推荐 |
| 治愈 | 包治、根治 |
| 虚假 | 错过等 1 年 |

### 8.3 安全建议

- ✅ 用"非常"、"很"、"真的"代替"最"
- ✅ 用"个人体验"代替"所有人都"
- ✅ 用"建议"代替"必须"
- ✅ 数据来源要标注
- ✅ 不夸大效果

## 九、算法监测指标

### 9.1 创作者后台核心指标

| 指标 | 健康值 | 优化方向 |
|------|--------|----------|
| 完播率 | ≥ 50% | 钩子 + 节奏 |
| 互动率 | ≥ 5% | CTA + 争议 |
| 关注转化 | ≥ 1% | 持续价值 |
| 推荐占比 | ≥ 70% | 内容质量 |
| 搜索占比 | ≥ 10% | 标题 + hashtag |

### 9.2 异常诊断

| 症状 | 可能原因 | 解决 |
|------|----------|------|
| 播放低 | 钩子弱 / 标题差 | 优化前 3 秒 |
| 完播低 | 节奏拖沓 | 删减内容 |
| 评论低 | CTA 缺失 | 加互动引导 |
| 关注低 | 价值感弱 | 加系列预告 |

## 十、WebPPT Maker 自动适配

### 10.1 自动化的部分

- ✅ 标题字数 ≤ 22
- ✅ 正文 80-150 字
- ✅ hashtag 5-8 个
- ✅ 开头 3s 钩子
- ✅ 结尾 CTA
- ✅ 时长 ± 2 秒

### 10.2 不能自动化的部分

- ❌ 真实数据来源（用户提供）
- ❌ 个人风格（用户偏好）
- ❌ 视觉素材（用户自选）
- ❌ 发布时间（用户决定）
