# 抖音文案模板

> 本文件由 `scripts/generate_copy.py` 读取并填充。
> 用户在 Claude Code 中调用时，agent 会读取本模板的格式，套用到用户给的内容上。

---

## 输出格式

```markdown
# {{TITLE}}

{{COPY_BODY}}

{{HASHTAGS}}

{{HOOKS}}
```

---

## 字段说明

### 1. TITLE（标题）
- **要求**：≤ 22 字（含 emoji）
- **风格**：强钩子 + 数字 + 好奇
- **示例**：
  - `AI 写作的 5 个真相，第 3 个扎心了`
  - `每天喝咖啡的 3 个真相，你肯定不知道`
  - `30 天转行 AI，我学到了这 5 件事`

### 2. COPY_BODY（正文）
- **要求**：80-150 字
- **节奏**：3-4 句一段，前 30 字必须勾住人
- **风格**：口语化、有节奏、含情绪
- **结构**：
  - 第 1 句：钩子（问题/反差/数字）
  - 第 2-3 句：展开（讲故事/列点）
  - 第 4 句：转折/高潮
  - 第 5 句：共鸣 + CTA 引导

### 3. HASHTAGS（话题标签）
- **数量**：5-8 个
- **类型**：
  - 1-2 个大流量：`#知识分享` `#AI` `#干货`
  - 2-3 个垂直领域：`#AI写作` `#职场干货`
  - 1-2 个趋势标签：`#2026爆款` `#热门话题`
- **格式**：每个 `#` 开头，空格分隔

### 4. HOOKS（互动钩子）
- **数量**：1-2 个
- **类型**：
  - 评论引导：`评论区告诉我` / `你怎么看`
  - 关注引导：`关注我，下期讲`
  - 收藏引导：`收藏起来慢慢看`
  - 分享引导：`@你的朋友一起来`

## 生成算法（脚本逻辑）

```python
def generate_copy(topic, points, platform="douyin"):
    # 1. 标题生成
    title = generate_title(topic, points)
    # 规则：字数 ≤ 22，含 emoji，含数字或钩子词

    # 2. 正文生成
    body = generate_body(topic, points)
    # 规则：80-150 字，3-5 句，钩子句在前 30 字

    # 3. Hashtag 生成
    hashtags = extract_hashtags(topic, points)
    # 规则：5-8 个，含大流量 + 垂直 + 趋势

    # 4. 钩子生成
    hooks = generate_hooks(topic, points)
    # 规则：1-2 个，含评论/关注/收藏引导

    return format_copy(title, body, hashtags, hooks)
```
